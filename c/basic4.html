C Programming: Basic Lesson 4 – Control Structures

Using if, else, and switch to control program flow

🔀 if-else Statements


int age = 18;

if (age >= 18) {
    printf("Adult\n");
} else {
    printf("Minor\n");
}

Checks a condition and runs code accordingly.

🔄 switch Statement


int day = 3;

switch(day) {
    case 1: printf("Monday\n"); break;
    case 2: printf("Tuesday\n"); break;
    case 3: printf("Wednesday\n"); break;
    default: printf("Invalid day\n");
}

Used for multi-way branching based on values.
