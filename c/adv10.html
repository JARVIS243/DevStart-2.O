Advanced C: Lesson 10 – Pointers and Functions

Pass data efficiently and safely using pointers

📌 Why Use Pointers in Functions?

  • To modify variables in the calling function
  • To pass large structures efficiently

🔁 Pass by Reference (Using Pointers)


#include <stdio.h>

void update(int *a) {
    *a = *a + 10;
}

int main() {
    int x = 5;
    update(&x);
    printf("Updated value: %d\n", x);
    return 0;
}

✅ Output: Updated value: 15

🔗 Pointer to Function


#include <stdio.h>

void greet() {
    printf("Hello from DevStart!\n");
}

int main() {
    void (*funcPtr)() = greet;
    funcPtr(); // calls greet()
    return 0;
}
