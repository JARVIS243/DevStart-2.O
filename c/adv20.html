🎓 Build a Library Management System in C

🎯 Project Goal

Design a Library Management System using:

  • Structures
  • File handling (read/write/search)
  • Functions & user menu

🧱 Book Structure


typedef struct {
    int id;
    char title[100];
    char author[50];
    int available;
} Book;

🧑‍💻 Main Features

  • Add a new book
  • List all books
  • Search by ID or title
  • Delete or update book details
  • Mark a book as borrowed/returned

🧪 Try Building This

  1. Create library.bin to store book records
  2. Use fwrite(), fread(), fseek()
  3. Design a clear CLI menu
  4. Use modular functions for each feature
