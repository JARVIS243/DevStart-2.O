C Programming: Intermediate Lesson 5 – Recursion

Functions calling themselves

🔄 What is Recursion?

A function that calls itself to solve smaller instances of a problem.

🧑‍💻 Example: Factorial Using Recursion


#include <stdio.h>

int factorial(int n) {
    if (n == 0) return 1;
    else return n * factorial(n - 1);
}

int main() {
    int num = 5;
    printf("Factorial of %d is %d\n", num, factorial(num));
    return 0;
}
