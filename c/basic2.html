C Programming: Basic Lesson 2 – Variables & Data Types

Understanding variables and basic data types in C

🔤 What is a Variable?

A variable is a named storage location in memory that holds a value which can be changed during program execution.

📚 Common Data Types in C

  • int – integer numbers (e.g., 10, -5)
  • float – single-precision floating point numbers (e.g., 3.14)
  • double – double-precision floating point numbers
  • char – single characters (e.g., 'A', 'z')
  • void – no data (used in functions)

🧑‍💻 Example: Declaring and Using Variables


#include <stdio.h>

int main() {
    int age = 21;
    float pi = 3.14;
    char initial = 'A';

    printf("Age: %d\n", age);
    printf("Pi: %.2f\n", pi);
    printf("Initial: %c\n", initial);

    return 0;
}

Explanation:

  • int age = 21; declares an integer variable age and assigns 21.
  • float pi = 3.14; declares a floating-point variable.
  • char initial = 'A'; declares a character variable.
  • printf uses format specifiers: %d for int, %f for float/double, %c for char.

🧪 Try This Yourself

  • Declare variables of each data type and print their values.
  • Experiment with changing variable values after declaration.
  • Try printing variables with incorrect format specifiers to see the result.
