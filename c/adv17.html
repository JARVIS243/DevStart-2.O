Advanced C: Lesson 17 – Macros & Preprocessor Directives

Use compile-time instructions to improve your code efficiency

⚙️ What is the Preprocessor?

Before compilation, the C preprocessor handles instructions like #define, #include, and #if.

🧾 Macros


#define PI 3.14159
#define SQUARE(x) ((x) * (x))

int main() {
    printf("PI: %f\n", PI);
    printf("Square of 5: %d\n", SQUARE(5)); // 25
    return 0;
}

⚠️ Use parentheses in macros to avoid bugs.

📂 Conditional Compilation


#define DEBUG

int main() {
    #ifdef DEBUG
        printf("Debugging enabled\n");
    #endif
    return 0;
}

📌 Other Preprocessor Directives

  • #include – add external files
  • #ifndef – check if not defined
  • #undef – remove a macro definition

🧪 Try This Yourself

  • Define a macro to convert Celsius to Fahrenheit
  • Use conditional compilation to hide/show test logs
