Advanced C: Lesson 1 – File Handling

Read, write, and append files using C

📄 Writing to a File


FILE *fp = fopen("notes.txt", "w");
fprintf(fp, "DevStart C File Handling Example\n");
fclose(fp);

📖 Reading from a File


FILE *fp = fopen("notes.txt", "r");
char line[100];
while (fgets(line, sizeof(line), fp)) {
    printf("%s", line);
}
fclose(fp);

➕ Appending to a File


fp = fopen("notes.txt", "a");
fprintf(fp, "This line is appended.\n");
fclose(fp);
