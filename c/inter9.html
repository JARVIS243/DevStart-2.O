C Programming: Intermediate Lesson 9 – Error Handling Basics

Handle runtime errors safely and effectively

To prevent crashes and bugs when things go wrong (e.g., missing file, memory failure).

📘 File Error Example


#include <stdio.h>

int main() {
    FILE *fp = fopen("data.txt", "r");

    if (fp == NULL) {
        printf("❌ Error: Could not open file!\n");
        return 1;
    }

    // Read file content...

    fclose(fp);
    return 0;
}

💾 Malloc Error Example


#include <stdio.h>
#include <stdlib.h>

int main() {
    int *ptr = (int *)malloc(1000000000000 * sizeof(int)); // very large

    if (ptr == NULL) {
        printf("❌ Error: Memory allocation failed!\n");
        return 1;
    }

    free(ptr);
    return 0;
}

🧪 Try This Yourself

  • Try to open a file that doesn’t exist.
  • Use malloc with a large size and check the result.
