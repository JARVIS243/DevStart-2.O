Advanced C: Lesson 12 – Command-Line Arguments

Pass values to your program through the terminal

🧾 What Are Command-Line Arguments?

Arguments passed to the program when it is run from the terminal.

Useful for dynamic input without using scanf or files.

🧑‍💻 Syntax


int main(int argc, char *argv[]) {
    // argc  - argument count
    // argv[] - argument values
}

💡 Example


#include <stdio.h>

int main(int argc, char *argv[]) {
    printf("Total arguments: %d\n", argc);
    for (int i = 0; i < argc; i++) {
        printf("Arg[%d]: %s\n", i, argv[i]);
    }
    return 0;
}

💬 Run it like: ./a.out Hello World
