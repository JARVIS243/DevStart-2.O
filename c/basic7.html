C Programming: Basic Lesson 7 – Arrays and Strings Basics

Understanding arrays and basic string handling

📚 Arrays

An array stores multiple values of the same type in contiguous memory.


int numbers[5] = {10, 20, 30, 40, 50};

🧑‍💻 Example: Accessing Array Elements


#include <stdio.h>

int main() {
    int numbers[3] = {5, 10, 15};
    printf("First element: %d\n", numbers[0]);
    printf("Second element: %d\n", numbers[1]);
    printf("Third element: %d\n", numbers[2]);
    return 0;
}

📖 Strings in C

Strings are arrays of characters ending with a null character '\0'.


char name[6] = {'A','j','u','\0'};
