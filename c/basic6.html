C Programming: Basic Lesson 6 – Functions

Learn how to write and use functions

🔧 What is a Function?

A reusable block of code that performs a specific task.

🧑‍💻 Example: Defining and Calling Functions


#include <stdio.h>

// Function declaration
void greet() {
    printf("Hello from a function!\n");
}

int main() {
    greet();  // Call the function
    return 0;
}

🔢 Functions with Parameters


#include <stdio.h>

void greetName(char name[]) {
    printf("Hello, %s!\n", name);
}

int main() {
    greetName("Aju");
    return 0;
}

🧪 Try This Yourself

  • Create functions that take different types of parameters.
  • Write functions that return values.
