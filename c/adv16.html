Advanced C: Lesson 16 – Bitwise Operators

Perform operations directly on bits using C

🔢 Common Bitwise Operators

  • & – AND
  • | – OR
  • ^ – XOR
  • ~ – NOT
  • << – Left Shift
  • >> – Right Shift

💡 Example


#include <stdio.h>

int main() {
    int a = 5, b = 3;

    printf("AND: %d\n", a & b);     // 1
    printf("OR: %d\n", a | b);      // 7
    printf("XOR: %d\n", a ^ b);     // 6
    printf("NOT a: %d\n", ~a);      // -6
    printf("a << 1: %d\n", a << 1); // 10
    printf("a >> 1: %d\n", a >> 1); // 2

    return 0;
}
