C Programming: Intermediate Lesson 1 – Advanced Pointers

Understanding pointer arithmetic, pointers and arrays

📍 Pointer Arithmetic

Pointers can be incremented/decremented to traverse arrays.


int arr[] = {10, 20, 30};
int *ptr = arr;

printf("%d\n", *ptr);     // 10
ptr++;
printf("%d\n", *ptr);     // 20

📚 Pointers and Arrays

Array names act as pointers to the first element.


int arr[3] = {1, 2, 3};
printf("%d\n", *(arr + 2));  // 3
