C Programming: Basic Lesson 1 – Introduction & Setup

Write your first C program: Hello, World!

👋 Why Learn C?

C is a powerful, efficient, and foundational programming language widely used in systems programming, embedded systems, and many applications.

🧑‍💻 Your First C Program


#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}

Explanation:

  • #include <stdio.h> includes the standard input-output library needed for printf.
  • int main() is the main function where execution starts.
  • printf() prints text to the screen.
  • return 0; ends the program and returns success.

🧪 Try This Yourself

  • Modify the program to print your name.
  • Print multiple lines using multiple printf statements.
  • Experiment with printing special characters like tabs \t and newlines \n.

🛠 Setting Up Your Environment

You can write and run C programs using:

  • Online compilers like OnlineGDB or Programiz
  • Install GCC (GNU Compiler Collection) on your computer
  • Use IDEs like Code::Blocks, Visual Studio Code with C extensions, or CLion
