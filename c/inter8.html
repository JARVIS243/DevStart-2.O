C Programming: Intermediate Lesson 8 – Multi-dimensional Arrays

Work with arrays of arrays (matrices)

📚 What are Multi-dimensional Arrays?

Arrays with more than one dimension, e.g., 2D arrays represent matrices.

🧑‍💻 Example: 2D Array


#include <stdio.h>

int main() {
    int matrix[2][3] = {
        {1, 2, 3},
        {4, 5, 6}
    };

    for (int i = 0; i < 2; i++) {
        for (int j = 0; j < 3; j++) {
            printf("%d ", matrix[i][j]);
        }
        printf("\n");
    }
    return 0;
}
