Advanced C: Lesson 8 – Searching Algorithms

Implement Linear and Binary Search in C

🔍 Linear Search


int linearSearch(int arr[], int n, int target) {
    for (int i = 0; i < n; i++) {
        if (arr[i] == target)
            return i;
    }
    return -1;
}

✅ Best for small or unsorted data

📈 Binary Search (Sorted Array Only)


int binarySearch(int arr[], int n, int target) {
    int low = 0, high = n - 1;
    while (low <= high) {
        int mid = (low + high) / 2;
        if (arr[mid] == target)
            return mid;
        else if (arr[mid] < target)
            low = mid + 1;
        else
            high = mid - 1;
    }
    return -1;
}

⚠️ Make sure the array is sorted before using Binary Search
