C Programming: Basic Lesson 5 – Loops

Using for, while, and do-while loops

🔁 for Loop


for (int i = 0; i < 5; i++) {
    printf("i = %d\n", i);
}

Repeats code a fixed number of times.

🌀 while Loop


int i = 0;
while (i < 5) {
    printf("i = %d\n", i);
    i++;
}

Repeats code while a condition is true.

🔄 do-while Loop


int i = 0;
do {
    printf("i = %d\n", i);
    i++;
} while (i < 5);

Executes code at least once, then repeats while condition true.
