C Programming: Basic Lesson 3 – Input and Output

Learn how to take input and display output in C

📥 Taking Input

Use scanf() to get input from the user.

📤 Displaying Output

Use printf() to print messages and variables to the screen.

🧑‍💻 Example: Input and Output


#include <stdio.h>

int main() {
    int age;
    char name[30];

    printf("Enter your age: ");
    scanf("%d", &age);

    printf("Enter your name: ");
    scanf("%s", name);

    printf("Hello %s, you are %d years old.\n", name, age);

    return 0;
}

Explanation:

  • scanf("%d", &age); reads an integer from user input.
  • scanf("%s", name); reads a string (word) without spaces.
  • Always use & before variable names for non-array inputs.

🧪 Try This Yourself

  • Take input for different data types and print them.
  • Try reading strings with spaces using other methods (later lessons).
