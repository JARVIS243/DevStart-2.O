C Programming: Intermediate Lesson 4 – File Handling Basics

Read from and write to files

📄 File Operations

Open, read, write, and close files using fopen, fread, fwrite, and fclose.

🧑‍💻 Example: Writing and Reading a File


#include <stdio.h>

int main() {
    FILE *fp;

    // Writing to file
    fp = fopen("data.txt", "w");
    if (fp == NULL) {
        printf("Error opening file\n");
        return 1;
    }
    fprintf(fp, "Hello, DevStart!\n");
    fclose(fp);

    // Reading from file
    char str[100];
    fp = fopen("data.txt", "r");
    if (fp == NULL) {
        printf("Error opening file\n");
        return 1;
    }
    while (fgets(str, 100, fp) != NULL) {
        printf("%s", str);
    }
    fclose(fp);

    return 0;
}
