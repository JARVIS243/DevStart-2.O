C Programming: Intermediate Lesson 10 – Typedef and Enums

Make your code cleaner and more readable

🔠 typedef

Creates an alias for a data type.


#include <stdio.h>

typedef unsigned int uint;

int main() {
    uint age = 25;
    printf("Age: %u\n", age);
    return 0;
}

🔢 enum (Enumeration)

Defines a set of named integer constants.


#include <stdio.h>

enum Day { MON, TUE, WED, THU, FRI };

int main() {
    enum Day today = WED;

    if (today == WED)
        printf("Mid-week day!\n");

    return 0;
}

🧪 Try This Yourself

  • Use typedef to simplify long data types (e.g., struct Student).
  • Create enums for months, levels, or categories.
