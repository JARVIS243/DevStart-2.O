C Programming: Intermediate Lesson 6 – Command-line Arguments

Pass arguments to programs via the command line

💻 What are Command-line Arguments?

Arguments passed to the program when it starts, accessible in main().

🧑‍💻 Example: Using argc and argv


#include <stdio.h>

int main(int argc, char *argv[]) {
    printf("Number of arguments: %d\n", argc);
    for (int i = 0; i < argc; i++) {
        printf("Argument %d: %s\n", i, argv[i]);
    }
    return 0;
}
