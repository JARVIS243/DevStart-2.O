Advanced C: Lesson 19 – Union in C

Use shared memory structure for efficient data handling

🔄 What is a Union?

A union is like a structure, but all members share the same memory location. Only one member can hold a value at any time.

💡 Example


#include <stdio.h>

union Data {
    int i;
    float f;
    char str[20];
};

int main() {
    union Data d;

    d.i = 10;
    printf("d.i = %d\n", d.i);

    d.f = 3.14;
    printf("d.f = %.2f\n", d.f);

    strcpy(d.str, "DevStart");
    printf("d.str = %s\n", d.str);

    return 0;
}

⚠️ Only the last assigned value remains valid.

🆚 Union vs Structure

  • Structure – separate memory for each member
  • Union – shared memory for all members
