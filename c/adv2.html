Advanced C: Lesson 2 – Bitwise Operators

Operate directly on bits for performance and control

🧠 Bitwise AND, OR, XOR


int a = 5, b = 3; // 0101 & 0011
printf("AND: %d\n", a & b);   // 1
printf("OR : %d\n", a | b);   // 7
printf("XOR: %d\n", a ^ b);   // 6

🔄 Shift Operators


int a = 4; // 0100
printf("Left Shift : %d\n", a << 1); // 8
printf("Right Shift: %d\n", a >> 1); // 2
