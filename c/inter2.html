C Programming: Intermediate Lesson 2 – Structures (struct)

Group related variables together

📦 What is a struct?

A struct groups variables of different types under one name.


struct Person {
    char name[50];
    int age;
    float height;
};

🧑‍💻 Example: Using structs


#include <stdio.h>

struct Person {
    char name[50];
    int age;
};

int main() {
    struct Person p1 = {"Aju", 21};
    printf("Name: %s\nAge: %d\n", p1.name, p1.age);
    return 0;
}
