Advanced C: Lesson 15 – Student Database Project

Mini project using structures, files, functions, and logic

🎯 Project Goal

Build a simple Student Database program that can:

  • Add student records
  • Display all records
  • Search a student by ID
  • Delete a student record

🧱 Structure Definition


struct Student {
    int id;
    char name[50];
    float grade;
};

🔄 Main Menu Logic


int main() {
    int choice;
    do {
        printf("1. Add\n2. View\n3. Search\n4. Delete\n5. Exit\n");
        scanf("%d", &choice);

        switch(choice) {
            case 1: addStudent(); break;
            case 2: viewAll(); break;
            case 3: searchById(); break;
            case 4: deleteById(); break;
        }
    } while (choice != 5);
    return 0;
}
