Kotlin – Basic 3: Input & Output

Learn how to get input from users and display output

📥 Taking User Input

Use readLine() to get user input from the console. It returns a String? (nullable string).


fun main() {
    print("Enter your name: ")
    val name = readLine()
    println("Hello, $name!")
}

🔁 Converting Input Types

Convert input from string to number types using methods like toInt() or toDouble().


fun main() {
    print("Enter your age: ")
    val ageInput = readLine()
    val age = ageInput?.toIntOrNull() ?: 0
    println("You are $age years old.")
}

⚠️ Use safe calls and null checks to avoid crashes if the user enters invalid input.

🖨️ Printing Output

  • print() – prints without a newline
  • println() – prints with a newline

fun main() {
    print("Hello ")
    println("World!")
}

Output:

Hello World!

🧠 Logic Tips

  • Always check if input is null or invalid
  • Use toIntOrNull() instead of toInt() to avoid exceptions
  • Use string templates like "Hello, $name" for cleaner code

  • Print a sentence like: Hello Aju, you like Kotlin!
  • Try entering numbers and adding them after converting
