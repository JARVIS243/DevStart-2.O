Kotlin – Advanced 6: Extension Functions

Add new functions to existing classes without inheritance

🧩 Example


fun String.reverse(): String {
    return this.reversed()
}

fun main() {
    println("DevStart".reverse()) // Output: traStveD
}

✅ Why Use Extension Functions?

  • Cleaner syntax for common operations
  • Add behavior to standard types (like String, Int)
