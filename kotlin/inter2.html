Kotlin – Intermediate 2: Constructors

Initialize object properties with constructors

🧱 Primary Constructor


class Student(val name: String, val age: Int) {
    fun display() {
        println("Name: $name, Age: $age")
    }
}

fun main() {
    val s1 = Student("Aju", 18)
    s1.display()
}

🧱 Secondary Constructor


class Book {
    var title: String
    var pages: Int

    constructor(title: String, pages: Int) {
        this.title = title
        this.pages = pages
    }

    fun info() {
        println("Book: $title, Pages: $pages")
    }
}

fun main() {
    val b1 = Book("DevStart Kotlin", 200)
    b1.info()
}

🧠 Logic Tips

  • Use val/var inside primary constructor to auto-create properties
  • this refers to the current object's property
  • Prefer primary constructor unless you need overloads or logic
