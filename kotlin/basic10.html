Add notes to your code for better understanding

💬 What are Comments?

Comments are ignored by the compiler. They're just for humans!


// This is a single-line comment

/*
This is a
multi-line comment
*/

fun main() {
    println("Hello") // prints Hello
}

🧠 Logic Tips

  • Use comments to explain logic or complex code
  • Clean code with clear variable names reduces the need for comments
  • Don’t overuse – comment only where it helps

🧪 Try This

  • Add a comment above every line of code you write today
  • Explain why you're using certain values or functions
