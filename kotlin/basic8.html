Kotlin – Basic 8: Lists & Arrays

Store multiple values using arrays and lists

📦 Array


fun main() {
    val numbers = arrayOf(10, 20, 30)
    println(numbers[0])
    println("Size: ${numbers.size}")
}

📚 List


fun main() {
    val fruits = listOf("Apple", "Banana", "Cherry")
    for (fruit in fruits) {
        println(fruit)
    }
}

listOf() creates an immutable list. Use mutableListOf() to add/remove items.

🧠 Logic Tips

  • Use arrayOf() when the size is fixed
  • Use listOf() for easy looping
  • Use mutableListOf() to modify the list
