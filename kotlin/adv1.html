Kotlin – Advanced 1: Lambda Expressions

Use anonymous functions to simplify logic

⚡ What is a Lambda?

A lambda expression is an anonymous function used to pass behavior as data.


fun main() {
    val greet = { name: String -> println("Hello, $name!") }
    greet("Aju")
}
      

// Lambda with return
val square: (Int) -> Int = { num -> num * num }
println(square(4))  // Output: 16

✅ Key Points

  • Syntax: { param -> body }
  • Can be assigned to variables and passed to functions
  • Used in functional programming (map, filter, etc.)
