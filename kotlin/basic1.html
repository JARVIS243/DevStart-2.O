Kotlin – Basic 1: Introduction & Setup

Start learning Kotlin from scratch, even with no prior programming knowledge.

📘 What is Kotlin?

Kotlin is a modern, concise, and safe programming language developed by JetBrains. It is the preferred language for Android app development and can also be used for backend, web, and multiplatform development.

  • ✅ Easy to read and write
  • ✅ 100% interoperable with Java
  • ✅ Official language for Android by Google

⚙️ Setting Up Kotlin

Option 1: Online Editor (Recommended for Beginners)

Use play.kotlinlang.org – No installation required.

Option 2: Install IntelliJ IDEA (for local development)

  1. Download IntelliJ IDEA Community Edition from JetBrains
  2. Create a new Kotlin Project
  3. Choose "Kotlin/JVM" as the project type

👨‍💻 Your First Kotlin Program


fun main() {
    println("Welcome to DevStart – Kotlin!")
}

🖨️ Output:


Welcome to DevStart – Kotlin!

🧠 Logic Tip

  • fun defines a function in Kotlin
  • main() is the starting point of any Kotlin program
  • println() prints text to the output

🧪 Try This

  • Change the text to print your own name
  • Use print() instead of println() and notice the difference
