Kotlin – Advanced 2: Higher-Order Functions

Functions that take other functions as arguments

🔁 Example


fun operate(a: Int, b: Int, op: (Int, Int) -> Int): Int {
    return op(a, b)
}

fun main() {
    val sum = operate(5, 3) { x, y -> x + y }
    println("Result: $sum") // Output: 8
}

✅ Why Use?

  • Powerful way to abstract behaviors
  • Useful for callbacks, event handling, collection operations
