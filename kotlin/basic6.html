Kotlin – Basic 6: Functions

Group your code into reusable blocks using functions

📦 What is a Function?

A function is a reusable block of code that performs a specific task.


fun greet() {
    println("Welcome to DevStart!")
}

fun main() {
    greet()
}

🔢 Function with Parameters


fun greetUser(name: String) {
    println("Hello, $name!")
}

fun main() {
    greetUser("Aju")
}

🔁 Function with Return Value


fun add(a: Int, b: Int): Int {
    return a + b
}

fun main() {
    val sum = add(5, 3)
    println("Sum is $sum")
}

🧠 Logic Tips

  • Functions help avoid repeating code
  • You can pass values (parameters) into functions
  • Use : Type to define the return type
