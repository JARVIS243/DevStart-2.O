Kotlin – Basic 2: Variables & Data Types

Learn how to declare variables and use different data types in Kotlin.

📦 What is a Variable?

A variable is used to store data like numbers, text, or true/false values.

Kotlin has two main ways to declare variables:

  • val – read-only (like a constant)
  • var – mutable (can change)

🧪 Examples


fun main() {
    val name = "DevStart"
    var age = 21

    println("Name: $name")
    println("Age: $age")

    age = 22  // okay because 'age' is declared using var
    // name = "Another" // ❌ Error! name is immutable (val)
}

🖨️ Output:


Name: DevStart
Age: 21

🔢 Common Data Types

  • String – text (e.g., "hello")
  • Int – whole numbers (e.g., 42)
  • Double – decimal numbers (e.g., 3.14)
  • Boolean – true or false
  • Char – single character (e.g., 'A')

🔎 Explicit Typing (Optional)


val language: String = "Kotlin"
val year: Int = 2025

Kotlin can often infer types automatically, so writing the type is optional unless needed.

🧠 Logic Tips

  • Use val whenever possible (better safety)
  • Use var only when the value needs to change
  • Try printing different types to get familiar

💡 Try This

  • Create a variable for your name, age, and a boolean for “are you a student?”
  • Print a message using all three
