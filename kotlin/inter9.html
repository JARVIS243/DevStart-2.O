Kotlin – Intermediate 9: Interfaces

Define a contract of methods without implementation

🧩 What is an Interface?

An interface defines methods but doesn't provide implementation (like a blueprint). Classes implement interfaces.


interface Drawable {
    fun draw()
}

class Circle : Drawable {
    override fun draw() {
        println("Drawing a circle")
    }
}

fun main() {
    val c = Circle()
    c.draw()
}

✅ Key Points

  • Use interface keyword
  • Use : InterfaceName to implement
  • Must override all abstract methods
