Kotlin – Intermediate 6: Data Classes

Simple classes that hold data

📦 What is a Data Class?

Data classes automatically generate useful functions like toString(), equals(), and copy().


data class User(val name: String, val age: Int)

fun main() {
    val u1 = User("Aju", 20)
    val u2 = u1.copy(age = 21)
    println(u1)
    println(u2)
}

✅ Key Points

  • Use data keyword
  • Must have at least one property in the primary constructor
  • Great for storing simple structured data
