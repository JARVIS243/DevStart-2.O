Kotlin – Basic 4: Conditional Statements

Make decisions in your program using if and when

🔁 if / else


fun main() {
    val score = 85

    if (score >= 90) {
        println("Excellent!")
    } else if (score >= 70) {
        println("Good Job!")
    } else {
        println("Keep Practicing!")
    }
}

🔀 when Statement


fun main() {
    val day = 3

    when (day) {
        1 -> println("Monday")
        2 -> println("Tuesday")
        3 -> println("Wednesday")
        else -> println("Unknown day")
    }
}

🧠 Logic Tips

  • if is great for checking ranges or comparisons
  • when is like switch-case, used for exact matches
  • You can return values directly using if or when

  • Use when to map numbers to months
