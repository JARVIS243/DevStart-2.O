Kotlin – Intermediate 7: Companion Objects

Static-like members using companion object

🧲 Companion Object Example


class MathUtil {
    companion object {
        fun square(n: Int): Int {
            return n * n
        }
    }
}

fun main() {
    println(MathUtil.square(5)) // 25
}

✅ Why Companion Objects?

  • Allows calling functions without creating objects
  • Behaves like static in other languages
