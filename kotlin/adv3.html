Kotlin – Advanced 3: Collections

Work with lists, sets, and maps

📚 Lists


fun main() {
    val nums = listOf(1, 2, 3, 4, 5)
    println(nums[0]) // 1

    val mutable = mutableListOf("A", "B")
    mutable.add("C")
    println(mutable)
}

🔢 Sets


val set = setOf(1, 2, 2, 3)
println(set) // [1, 2, 3]

🗺️ Maps


val map = mapOf("Aju" to 18, "Dev" to 21)
println(map["Aju"]) // 18

✅ Key Points

  • listOf, setOf, mapOf = immutable
  • mutableListOf, etc. = can be changed
