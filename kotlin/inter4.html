Kotlin – Intermediate 4: Overriding

Change the behavior of a function in a subclass

🔁 Overriding a Method


open class Animal {
    open fun sound() {
        println("Some sound")
    }
}

class Cat : Animal() {
    override fun sound() {
        println("Meow")
    }
}

fun main() {
    val c = Cat()
    c.sound() // Meow
}

🧠 Logic Tips

  • Use open to allow a function to be overridden
  • Use override in subclass
  • This allows customizing behavior for specific child types
