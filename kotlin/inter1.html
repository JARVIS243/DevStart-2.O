🏗️ What is a Class?

A class is a blueprint for creating objects. Each object is an instance of a class.


class Car {
    var brand = "Tesla"
    var year = 2024

    fun drive() {
        println("Driving $brand")
    }
}

fun main() {
    val myCar = Car()
    println(myCar.brand)
    myCar.drive()
}

🧠 Logic Tips

  • A class defines properties and behaviors
  • Use val obj = ClassName() to create an object
  • Use dot (.) to access properties/functions
