Kotlin – Intermediate 3: Inheritance

Reusing code using parent and child classes

📦 Superclass and Subclass


open class Animal {
    fun eat() {
        println("Eating...")
    }
}

class Dog : Animal() {
    fun bark() {
        println("Barking...")
    }
}

fun main() {
    val dog = Dog()
    dog.eat() // Inherited
    dog.bark()
}

🧠 Logic Tips

  • Use open keyword to allow a class to be inherited
  • : is used to inherit from a superclass
  • Subclass gets all accessible functions from parent
