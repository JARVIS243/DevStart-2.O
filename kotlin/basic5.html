Kotlin – Basic 5: Loops

Repeat tasks using for and while loops

🔁 for Loop


fun main() {
    for (i in 1..5) {
        println("Count: $i")
    }
}

🔄 while Loop


fun main() {
    var i = 1
    while (i <= 5) {
        println("While count: $i")
        i++
    }
}

🔃 Loop Over List


fun main() {
    val fruits = listOf("Apple", "Banana", "Cherry")

    for (fruit in fruits) {
        println(fruit)
    }
}

🧠 Logic Tips

  • for is best when you know how many times to loop
  • while is useful when condition-based looping is needed
  • Use ranges (1..5) for cleaner loops

🧪 Try This

  • Print even numbers from 1 to 20 using for
  • Keep asking for a number until user enters 0 using while
