Kotlin – Advanced 10: Reflection

Inspect and manipulate code at runtime

🔍 What is Reflection?

Reflection lets you examine and interact with classes, methods, and properties during runtime.


import kotlin.reflect.full.memberProperties

data class User(val name: String, val age: Int)

fun main() {
    val user = User("Aju", 20)
    val kClass = user::class

    for (prop in kClass.memberProperties) {
        println("${prop.name} = ${prop.get(user)}")
    }
}

✅ Key Uses

  • Frameworks (serialization, DI, testing)
  • Accessing annotations, types, values
  • Debugging and tooling

⚠️ Note

Reflection may slow performance. Use wisely!
