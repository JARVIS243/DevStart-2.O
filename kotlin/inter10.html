Kotlin – Intermediate 10: Abstract Classes

Define base classes with optional or required behavior

🧱 What is an Abstract Class?

An abstract class cannot be instantiated. It can have abstract (unimplemented) and concrete (implemented) methods.


abstract class Animal {
    abstract fun makeSound()
    fun sleep() {
        println("Sleeping...")
    }
}

class Dog : Animal() {
    override fun makeSound() {
        println("Woof!")
    }
}

fun main() {
    val dog = Dog()
    dog.makeSound()
    dog.sleep()
}

✅ Key Points

  • Use abstract keyword
  • Can contain both abstract and regular methods
  • Must override abstract methods in subclasses
