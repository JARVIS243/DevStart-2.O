Kotlin – Basic 9: Strings

Work with text using string functions and templates

🔤 String Basics


fun main() {
    val name = "DevStart"
    val greeting = "Hello, $name!"
    println(greeting)
    println("Length: ${name.length}")
}

🔧 Useful String Functions


val text = "  Kotlin Programming  "

println(text.trim())      // removes extra spaces
println(text.uppercase()) // to UPPERCASE
println(text.lowercase()) // to lowercase
println(text.contains("Pro")) // true

🧠 Logic Tips

  • Use $ inside strings for easy variable insertion
  • Trim and clean inputs before using them
  • Compare strings using equals() or ==
