HTML & CSS – Intermediate 6: Transitions

Animate changes smoothly using CSS transitions

🎨 Hover Example


.btn {
  transition: background 0.3s ease;
}

.btn:hover {
  background: darkblue;
}

  • Create a button that changes color when hovered
