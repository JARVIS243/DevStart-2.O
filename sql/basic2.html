SQL – Basic 2: Creating a Database

Learn how to create a database using SQL

🛠️ SQL Syntax


CREATE DATABASE School;

This creates a new database named School.

✅ Tips

  • Database names should be unique
  • Use DROP DATABASE School; to delete it (careful!)
