SQL – Advanced 2: Indexes

Speed up query performance with indexes

⚡ Syntax


CREATE INDEX idx_name
ON Students (name);

This creates an index on the name column of the Students table.

📌 Notes

  • Indexes improve SELECT query speed
  • Can slow down INSERT/UPDATE/DELETE slightly

🧪 Try This

  • Create index on grade column
  • Test performance with and without index (optional)
