SQL – Intermediate 2: IN, BETWEEN, LIKE

Use pattern matching and set/interval filtering

🔍 IN


SELECT * FROM Students
WHERE grade IN ('A', 'B');

🔢 BETWEEN


SELECT * FROM Students
WHERE age BETWEEN 15 AND 17;

🔡 LIKE


SELECT * FROM Students
WHERE name LIKE 'J%';

% means wildcard. 'J%' = starts with J

🧪 Try This

  • List students with grade C or D using IN
  • Find students aged 14 to 18 using BETWEEN
  • Find names ending with “n” using LIKE
