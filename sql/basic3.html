SQL – Basic 3: Creating Tables

Design and create tables to store data in your database

📐 SQL Table Example


CREATE TABLE Students (
  id INTEGER PRIMARY KEY,
  name TEXT NOT NULL,
  age INTEGER,
  grade TEXT
);

This creates a table called Students with four columns.

✅ Column Types

  • INTEGER – whole numbers
  • TEXT – strings/characters
  • PRIMARY KEY – unique ID
  • NOT NULL – field must have a value
