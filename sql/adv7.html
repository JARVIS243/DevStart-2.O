SQL – Advanced 7: Backup & Restore

Protect your database by saving and restoring data

💾 Backup (MySQL CLI)


mysqldump -u root -p devstart_db > backup.sql

📂 Restore


mysql -u root -p devstart_db < backup.sql

Use these commands to manually save and reload database content.

🧪 Try This

  • Backup your current DB using mysqldump
  • Delete DB and restore it from backup
