SQL – Advanced 5: Triggers

Automatically run SQL code after events


CREATE TRIGGER log_insert
AFTER INSERT ON Students
FOR EACH ROW
INSERT INTO Logs(message)
VALUES('A new student was added.');

Automatically logs activity when new data is inserted.
