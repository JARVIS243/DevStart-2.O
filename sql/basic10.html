SQL – Basic 10: LIMIT & DISTINCT

Control the number of rows returned and remove duplicates

🔢 LIMIT


SELECT * FROM Students
LIMIT 5;

Returns only the first 5 rows.

🎯 DISTINCT


SELECT DISTINCT grade FROM Students;

Returns only unique grade values.
