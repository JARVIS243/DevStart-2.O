SQL – Intermediate 5: Aliases

Rename columns or tables for better readability

📝 Column Aliases


SELECT name AS student_name, grade AS result
FROM Students;

AS gives a new temporary name to columns.

🧾 Table Aliases


SELECT s.name FROM Students AS s;
