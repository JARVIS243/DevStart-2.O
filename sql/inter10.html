SQL – Intermediate 10: Subqueries

Use queries inside other queries

🔍 Example


SELECT name FROM Students
WHERE age = (
  SELECT MAX(age) FROM Students
);

This returns the student with the highest age.

💡 When to Use

  • Compare values with aggregates (MIN, MAX, AVG)
  • Filter using another query
  • Build dynamic results

🧪 Try This

  • Find student with the lowest grade using a subquery
  • List students older than average age
