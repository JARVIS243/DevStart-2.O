SQL – Intermediate 9: FULL OUTER JOIN

Returns all records when there is a match in either left or right table

🔗 Syntax


SELECT Students.name, Marks.subject
FROM Students
FULL OUTER JOIN Marks ON Students.id = Marks.student_id;

Includes all students and all subjects, even if there's no match.

Note: FULL OUTER JOIN may not be supported in some SQL engines like MySQL.

🧪 Try This

  • Show all students and their subjects (including unmatched)
  • Identify rows with missing match on either side
