SQL – Advanced 4: Stored Procedures

Save reusable SQL code on the database

📦 Example (MySQL)


DELIMITER //
CREATE PROCEDURE GetAllStudents()
BEGIN
  SELECT * FROM Students;
END //
DELIMITER ;

▶️ Run It


CALL GetAllStudents();
