SQL – Advanced 1: Views

Save a query as a virtual table

🧱 Creating a View


CREATE VIEW TopStudents AS
SELECT name, grade FROM Students
WHERE grade = 'A';

🔍 Using a View


SELECT * FROM TopStudents;

🧪 Try This

  • Create a view for students with grade B or above
  • Query the view to list names only
