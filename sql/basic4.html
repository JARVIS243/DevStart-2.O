SQL – Basic 4: Inserting Data

Learn how to insert records into a table

✏️ Syntax


INSERT INTO Students (id, name, age, grade)
VALUES (1, 'John Doe', 17, 'A');

This command adds a new student to the Students table.

🧠 Tips

  • Use single quotes for text values
  • Match column order with value order

🧪 Try This

  • Insert 3 students with different grades
  • Try inserting without some optional fields
