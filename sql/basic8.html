SQL – Basic 8: DELETE Statement

Remove data from your database

🗑️ Syntax


DELETE FROM Students
WHERE id = 2;

This will delete the student with ID = 2.

⚠️ Warning!

If you forget the WHERE clause, all rows will be deleted!


-- Deletes ALL records
DELETE FROM Students;

🧪 Try This

  • Delete one student by ID
  • Delete all students with grade 'F'
