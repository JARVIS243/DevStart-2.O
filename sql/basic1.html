SQL – Basic 1: What is SQL?

Learn what SQL is and how it's used to manage data.

📘 What is SQL?

SQL (Structured Query Language) is the standard language used to communicate with relational databases.

  • Query and retrieve data
  • Insert, update, and delete records
  • Create and manage tables

🗄️ Where is SQL Used?

SQL is used in databases like MySQL, PostgreSQL, SQLite, SQL Server, Oracle, etc.

🧠 Try This

Install SQLite or use an online SQL compiler like sqliteonline.com.
