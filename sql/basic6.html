SQL – Basic 6: WHERE Clause

Filter your query results using conditions

🔍 Syntax


SELECT * FROM Students
WHERE grade = 'A';

Returns only students who have grade A.

🧠 You Can Use

  • =, >, <, >=, <=
  • != or <> (not equal)

🧪 Try This

  • Find students older than 16
  • List students with grade not equal to 'B'
