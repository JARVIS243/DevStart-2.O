SQL – Basic 7: UPDATE

Change values in an existing record

✏️ Syntax


UPDATE Students
SET grade = 'A+'
WHERE id = 1;

This changes the grade of student with ID 1 to A+.

⚠️ Be Careful!

If you omit the WHERE clause, all rows will be updated!


-- Dangerous!
UPDATE Students SET grade = 'F';

🧪 Try This

  • Update the name of a specific student
  • Change grades for students with grade = 'B'
