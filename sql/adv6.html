SQL – Advanced 6: User Permissions

Control what different users can do

👤 Create User


CREATE USER 'devuser'@'localhost' IDENTIFIED BY 'password123';

🔐 Grant Permissions


GRANT SELECT, INSERT ON Students TO 'devuser'@'localhost';

🚫 Revoke


REVOKE INSERT ON Students FROM 'devuser'@'localhost';

🧪 Try This

  • Create a read-only user
  • Revoke DELETE permission from admin user
