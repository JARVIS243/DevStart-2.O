SQL – Intermediate 8: RIGHT JOIN

Include all records from the right table and matched rows from the left

📘 Syntax


SELECT Students.name, Marks.subject
FROM Students
RIGHT JOIN Marks ON Students.id = Marks.student_id;

Returns all marks entries, even if no student matches.

🧪 Try This

  • List all subjects and the student who took them (if any)
  • Show subjects without student records
