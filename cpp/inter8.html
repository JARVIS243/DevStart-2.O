C++: Intermediate Lesson 8 – Templates

Write generic and reusable code using templates

📚 What are Templates?

Templates allow you to write functions and classes that work with any data type.

🧑‍💻 Example: Function Template


#include <iostream>
using namespace std;

template <typename T>
T add(T a, T b) {
    return a + b;
}

int main() {
    cout << add(5, 3) << endl;        // int
    cout << add(2.5, 3.1) << endl;    // double
    return 0;
}

🔍 Notes

  • Templates help avoid code duplication
  • Works with functions and classes
  • Use template <typename T> syntax
