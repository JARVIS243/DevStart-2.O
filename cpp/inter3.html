C++: Intermediate Lesson 3 – References and Aliases

Understand references as alternative names for variables

🔗 What is a Reference?

A reference is an alias for another variable. It must be initialized when declared.

🧑‍💻 Example: Declaring and Using References


#include <iostream>
using namespace std;

int main() {
    int original = 10;
    int &ref = original; // ref is reference to original

    cout << "original = " << original << endl;
    cout << "ref = " << ref << endl;

    ref = 20; // changes original
    cout << "After modifying ref:" << endl;
    cout << "original = " << original << endl;
    cout << "ref = " << ref << endl;

    return 0;
}

🔍 Notes

  • References must be initialized at declaration
  • They cannot be null or changed to refer to another variable
  • Useful for passing variables to functions without copying
