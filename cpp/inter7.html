C++: Intermediate Lesson 7 – Polymorphism

Use the same interface for different data types

🌀 What is Polymorphism?

Polymorphism allows functions or objects to behave differently based on the context.

🧑‍💻 Example: Virtual Functions


#include <iostream>
using namespace std;

class Animal {
public:
    virtual void speak() {
        cout << "Animal speaks" << endl;
    }
};

class Dog : public Animal {
public:
    void speak() override {
        cout << "Dog barks" << endl;
    }
};

int main() {
    Animal* animal = new Dog();
    animal->speak(); // calls Dog's speak()

    delete animal;
    return 0;
}

🔍 Notes

  • Use virtual keyword for functions to allow overriding
  • Derived class functions use override keyword for clarity
  • Polymorphism enables dynamic method dispatch

🧪 Try This

  • Create a base class and derived classes with overridden methods
  • Use pointers or references to call methods polymorphically
  • Experiment with virtual destructors
