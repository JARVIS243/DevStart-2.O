C++: Intermediate Lesson 1 – Pointers Basics

Understand pointers and how to use them

📍 What is a Pointer?

A pointer is a variable that stores the memory address of another variable.

🧑‍💻 Example: Declaring and Using Pointers


#include <iostream>
using namespace std;

int main() {
    int var = 42;
    int* ptr = &var;  // ptr stores address of var

    cout << "Value of var: " << var << endl;
    cout << "Address of var: " << &var << endl;
    cout << "Pointer ptr points to address: " << ptr << endl;
    cout << "Value pointed by ptr: " << *ptr << endl;

    return 0;
}

🔍 Notes

  • * is used to declare a pointer and to dereference it (get value at address)
  • & is the address-of operator
  • Pointers must be of the correct type
