C++: Basic Lesson 10 – Practice Exercises

Apply what you learned with these exercises

📝 Exercises

  1. Write a program to print "Hello, World!"
  2. Declare variables of different types and print their values
  3. Take user input for name and age, then print a greeting
  4. Write a program that uses if-else to check if a number is positive or negative
  5. Write a loop to print numbers from 1 to 10
  6. Create a function that returns the square of a number and call it
  7. Declare an array of 5 integers and print them
  8. Take two strings as input and concatenate them

💡 Tips

  • Try writing the code yourself before looking up answers
  • Use comments to organize your code
  • Test your programs thoroughly

🎯 Goal

Build confidence in C++ basics through practice and repetition.
