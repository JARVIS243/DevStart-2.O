C++: Basic Lesson 6 – Loops

Repeat actions using while, for, and do-while loops

🔄 while Loop


int i = 1;
while (i <= 5) {
    cout << i << " ";
    i++;
}
cout << endl;

Prints numbers 1 to 5.

🔁 for Loop


for (int i = 1; i <= 5; i++) {
    cout << i << " ";
}
cout << endl;

Another way to print numbers 1 to 5.

🔂 do-while Loop


int i = 1;
do {
    cout << i << " ";
    i++;
} while (i <= 5);
cout << endl;

Executes the loop body at least once, then checks the condition.
