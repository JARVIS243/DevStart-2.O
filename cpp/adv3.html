C++: Advanced Lesson 3 – Lambda Expressions

Write inline anonymous functions easily

📚 What is a Lambda Expression?

A lambda is an anonymous function you can define inline, useful for short snippets of code.

🧑‍💻 Example: Basic Lambda Usage


#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    vector<int> nums = {5, 3, 8, 1};

    // Sort in ascending order using lambda
    sort(nums.begin(), nums.end(), [](int a, int b) {
        return a < b;
    });

    for (int n : nums) {
        cout << n << " ";
    }
    cout << endl;

    // Lambda with captured variable
    int factor = 2;
    auto multiply = [factor](int x) {
        return x * factor;
    };
    cout << multiply(5) << endl;  // Output: 10

    return 0;
}

🔍 Notes

  • Lambdas can capture variables from their surrounding scope
  • Useful for callbacks, short functions, and algorithms
  • Syntax: [captures](parameters) { body }

🧪 Try This

  • Write lambdas for filtering and transforming data
  • Experiment with capture modes (= and &)
  • Use lambdas with STL algorithms like for_each
