C++: Basic Lesson 7 – Functions

Organize code into reusable blocks called functions

🔹 What is a Function?

A function is a block of code that performs a specific task and can be called multiple times.

🧑‍💻 Example: Function Definition and Call


#include <iostream>
using namespace std;

// Function declaration and definition
void greet() {
    cout << "Hello from function!" << endl;
}

int main() {
    greet();  // Call the function
    return 0;
}

📘 Function with Parameters and Return Value


int add(int a, int b) {
    return a + b;
}

int main() {
    int sum = add(5, 7);
    cout << "Sum: " << sum << endl;
    return 0;
}
