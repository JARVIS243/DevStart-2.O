C++: Intermediate Lesson 9 – Exception Handling

Handle errors gracefully using try, catch, and throw

⚠️ What is Exception Handling?

Mechanism to catch and handle runtime errors to prevent program crashes.

🧑‍💻 Example: Basic try-catch


#include <iostream>
using namespace std;

int divide(int a, int b) {
    if (b == 0) {
        throw "Division by zero error!";
    }
    return a / b;
}

int main() {
    try {
        cout << divide(10, 2) << endl;
        cout << divide(10, 0) << endl;
    }
    catch (const char* e) {
        cout << "Exception caught: " << e << endl;
    }

    return 0;
}

🔍 Notes

  • Use throw to raise exceptions
  • try block contains code that might throw
  • catch block handles exceptions

🧪 Try This

  • Throw exceptions of different types (int, string)
  • Write nested try-catch blocks
  • Create custom exception classes
