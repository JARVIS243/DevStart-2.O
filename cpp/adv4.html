C++: Advanced Lesson 4 – Smart Pointers

Manage dynamic memory safely using smart pointers

🧠 What are Smart Pointers?

Smart pointers are objects that manage the lifetime of dynamically allocated memory automatically.

🧑‍💻 Example: Using std::unique_ptr


#include <iostream>
#include <memory>
using namespace std;

class Demo {
public:
    Demo() { cout << "Constructor called\n"; }
    ~Demo() { cout << "Destructor called\n"; }
    void greet() { cout << "Hello from Demo\n"; }
};

int main() {
    unique_ptr<Demo> ptr = make_unique<Demo>();
    ptr->greet();

    // No need to delete ptr manually, destructor called automatically

    return 0;
}

🔍 Notes

  • unique_ptr owns the object exclusively
  • shared_ptr allows multiple owners
  • weak_ptr prevents cyclic references
  • Prevents memory leaks and dangling pointers
