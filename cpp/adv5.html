C++: Advanced Lesson 5 – Multithreading Basics

Learn how to run multiple threads concurrently

⚙️ What is Multithreading?

Multithreading allows your program to run multiple sequences of instructions simultaneously.

🧑‍💻 Example: Basic Thread Creation


#include <iostream>
#include <thread>

using namespace std;

void printMessage(string msg) {
    cout << msg << endl;
}

int main() {
    thread t1(printMessage, "Hello from thread 1");
    thread t2(printMessage, "Hello from thread 2");

    // Wait for threads to finish
    t1.join();
    t2.join();

    return 0;
}

🔍 Notes

  • Use std::thread to create threads
  • Use join() to wait for thread completion
  • Ensure thread-safe access to shared resources

🧪 Try This

  • Create threads that modify shared variables
  • Use mutexes to prevent data races
  • Experiment with thread synchronization techniques
