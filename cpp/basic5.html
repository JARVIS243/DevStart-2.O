C++: Basic Lesson 5 – Conditional Statements

Make decisions in your program using if, else, and switch

🔍 if-else Statements


int number = 10;

if (number > 0) {
    cout << "Positive number" << endl;
} else if (number == 0) {
    cout << "Zero" << endl;
} else {
    cout << "Negative number" << endl;
}

📘 switch Statement


char grade = 'B';

switch (grade) {
    case 'A':
        cout << "Excellent!" << endl;
        break;
    case 'B':
        cout << "Well done" << endl;
        break;
    case 'C':
        cout << "Good" << endl;
        break;
    default:
        cout << "Try harder" << endl;
}

🧪 Try This

  • Write a program to check if a number is even or odd
  • Use switch to print the day of the week given a number (1–7)

📌 Summary

  • if checks a condition and runs code if true
  • else runs code if the if condition is false
  • switch chooses between many options based on a variable
