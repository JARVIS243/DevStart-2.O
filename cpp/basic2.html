C++: Basic Lesson 2 – Variables and Data Types

Store and manage data using variables

📦 What is a Variable?

A variable is a name given to a memory location where you can store data. Each variable has a type, which defines the kind of data it can hold.

📘 Common Data Types

  • int – integers (e.g., 5, -2)
  • float – decimal numbers (e.g., 3.14)
  • char – single character (e.g., 'A')
  • bool – true or false
  • string – text (need #include <string>)

🧑‍💻 Example


#include <iostream>
#include <string>
using namespace std;

int main() {
    int age = 18;
    float height = 5.8;
    char grade = 'A';
    bool passed = true;
    string name = "DevStart";

    cout << "Name: " << name << endl;
    cout << "Age: " << age << endl;
    cout << "Height: " << height << endl;
    cout << "Grade: " << grade << endl;
    cout << "Passed: " << passed << endl;

    return 0;
}

🔍 Notes

  • Use = to assign values
  • Use << with cout to print values
  • bool prints as 1 (true) or 0 (false)

🧪 Try This

  • Declare your own variables (e.g., country, marks, isPassed)
  • Change the values and see the output
  • Try assigning a float to an int and observe what happens
