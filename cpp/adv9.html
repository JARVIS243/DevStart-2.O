C++: Advanced Lesson 9 – Coroutines

Write asynchronous code using coroutines

🧠 What are Coroutines?

Coroutines allow functions to be paused and resumed, enabling asynchronous programming.

🧑‍💻 Basic Example (C++20)


// Requires C++20 support

#include <coroutine>
#include <iostream>
using namespace std;

struct Task {
    struct promise_type {
        Task get_return_object() { return {}; }
        suspend_never initial_suspend() { return {}; }
        suspend_never final_suspend() noexcept { return {}; }
        void return_void() {}
        void unhandled_exception() {}
    };
};

Task exampleCoroutine() {
    cout << "Start coroutine\n";
    co_await std::suspend_always{};
    cout << "Resume coroutine\n";
}

int main() {
    auto task = exampleCoroutine();
    cout << "Main function continues\n";
    // Coroutine suspended at co_await and can be resumed later
    return 0;
}

🔍 Notes

  • Coroutines simplify asynchronous code
  • Introduced in C++20 standard
  • Requires compiler support and special syntax
