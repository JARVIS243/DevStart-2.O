Learn Object-Oriented Programming basics with classes and objects

🏛️ What is a Class?

A class is a blueprint for creating objects. It defines attributes (data) and behaviors (functions).

🧑‍💻 Example: Defining a Class and Creating an Object


#include <iostream>
using namespace std;

class Car {
public:
    string brand;
    int year;

    void displayInfo() {
        cout << "Brand: " << brand << ", Year: " << year << endl;
    }
};

int main() {
    Car myCar;
    myCar.brand = "Toyota";
    myCar.year = 2020;
    myCar.displayInfo();

    return 0;
}

🔍 Notes

  • Use class keyword to define a class
  • Members can be public or private (default private)
  • Objects are instances of classes
