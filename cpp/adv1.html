C++: Advanced Lesson 1 – Standard Template Library (STL): Vectors

Learn dynamic arrays using STL vectors

📚 What is a Vector?

Vectors are dynamic arrays that can change size during runtime.

🧑‍💻 Example: Basic Vector Usage


#include <iostream>
#include <vector>
using namespace std;

int main() {
    vector<int> numbers;

    // Add elements
    numbers.push_back(10);
    numbers.push_back(20);
    numbers.push_back(30);

    // Access elements
    for (int i = 0; i < numbers.size(); i++) {
        cout << "Element " << i << ": " << numbers[i] << endl;
    }

    return 0;
}

🔍 Notes

  • Vectors handle memory automatically
  • Use push_back() to add elements
  • Access elements using [] or at()
