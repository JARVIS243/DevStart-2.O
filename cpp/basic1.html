C++: Basic Lesson 1 – What is C++?

Write your first C++ program and understand how it works

💡 What is C++?

C++ is a powerful, high-performance programming language used in systems software, game development, real-time applications, and more. It supports both procedural and object-oriented programming.

🧑‍💻 Your First C++ Program


// File: main.cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello, DevStart!" << endl;
    return 0;
}

🔍 Explanation

  • #include <iostream> – allows input/output
  • using namespace std; – simplifies syntax
  • main() – starting point of the program
  • cout – prints output to the screen
  • return 0; – ends the program successfully

🧪 Try This

  • Change the message to your name
  • Print multiple lines using \n
  • Remove using namespace std; and see what happens

📌 Summary

  • Every C++ program starts with a main() function
  • Use cout for output, cin for input (coming next!)
  • Header files like <iostream> bring built-in functionality
