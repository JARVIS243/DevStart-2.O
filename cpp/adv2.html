C++: Advanced Lesson 2 – Standard Template Library (STL): Maps

Learn associative containers with key-value pairs

📚 What is a Map?

Maps store elements as key-value pairs and allow fast retrieval by key.

🧑‍💻 Example: Basic Map Usage


#include <iostream>
#include <map>
using namespace std;

int main() {
    map<string, int> ageMap;

    // Insert key-value pairs
    ageMap["Aju"] = 21;
    ageMap["Deva"] = 22;
    ageMap["Rani"] = 20;

    // Access elements
    for (auto &entry : ageMap) {
        cout << entry.first << " is " << entry.second << " years old." << endl;
    }

    return 0;
}

🔍 Notes

  • Maps store unique keys with associated values
  • Keys are sorted automatically
  • Access values using the key
