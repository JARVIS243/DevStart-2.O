C++: Basic Lesson 9 – Strings

Handle text using the string class

📜 What is a String?

In C++, a string is a sequence of characters. The string class from the standard library makes working with text easier than using character arrays.

🧑‍💻 Example: Using Strings


#include <iostream>
#include <string>
using namespace std;

int main() {
    string name = "DevStart";
    cout << "Hello, " << name << "!" << endl;

    // Getting input
    string userName;
    cout << "Enter your name: ";
    getline(cin, userName);  // reads full line including spaces

    cout << "Welcome, " << userName << "!" << endl;

    return 0;
}

🔍 Notes

  • Include <string> header
  • Use getline(cin, var) to read full lines including spaces
  • Strings support many useful functions like length(), substr(), and more
