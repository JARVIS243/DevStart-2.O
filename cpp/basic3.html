C++: Basic Lesson 3 – Input and Output

Take input from the user and display output on the screen

🧾 Output with cout

cout is used to print values to the screen.


#include <iostream>
using namespace std;

int main() {
    cout << "Welcome to DevStart!" << endl;
    cout << "Let's learn C++" << endl;
    return 0;
}

🎤 Input with cin

cin is used to take input from the user via keyboard.


#include <iostream>
using namespace std;

int main() {
    int age;
    cout << "Enter your age: ";
    cin >> age;
    cout << "You are " << age << " years old." << endl;
    return 0;
}

🔁 Multiple Inputs

You can accept multiple inputs in a single line.


int a, b;
cin >> a >> b;

This reads two numbers separated by space.

🧪 Try This

  • Take name, age, and country as input
  • Display a custom greeting using the values
  • Try entering characters and decimals into an int variable

📌 Summary

  • cout – output stream to print to screen
  • cin – input stream to read from user
  • Use << with cout, >> with cin
