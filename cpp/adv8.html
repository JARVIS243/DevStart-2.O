C++: Advanced Lesson 8 – Concepts

Use constraints to specify template requirements

📚 What are Concepts?

Concepts allow you to specify requirements for template parameters, improving code clarity and error messages.

🧑‍💻 Example: Using Concepts


#include <concepts>
#include <iostream>
using namespace std;

template <typename T>
concept Incrementable = requires(T x) {
    x++;
};

template <Incrementable T>
T increment(T value) {
    return value + 1;
}

int main() {
    cout << increment(5) << endl;
    // cout << increment("Hello") << endl; // Error: string not Incrementable

    return 0;
}

🔍 Notes

  • Concepts improve template code readability
  • They specify what operations a type must support
  • Helps catch template errors at compile time
