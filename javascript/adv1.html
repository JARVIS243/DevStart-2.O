JavaScript: Advanced Lesson 1 – Functions

Write reusable blocks of code using functions

🧠 What is a Function?

A function is a block of code that performs a specific task. It can be reused, called multiple times, and helps break your program into smaller parts.

✍️ Declaring and Calling a Function


function greet() {
  console.log("Hello, DevStart learner!");
}

greet(); // calling the function

- function keyword is used to define a function - greet() is the function name - The code inside runs only when the function is called

📥 Functions with Parameters


function greetUser(name) {
  console.log("Welcome, " + name + "!");
}

greetUser("Aju");
greetUser("DevStart");

- name is a parameter (input) - You can pass any value while calling the function

🔁 Functions That Return Values


function add(a, b) {
  return a + b;
}

let result = add(5, 3);  // result = 8
console.log(result);

- return sends a value back to where the function was called - You can store the returned result in a variable

⚠️ Why Use Functions?

  • Reusability – write once, use many times
  • Organized code – divide large logic into steps
  • Improves readability – name describes purpose

🧪 Try This

  • Create a function that multiplies 2 numbers
  • Make a function that returns if a number is even or odd
  • Write a function to greet a user by name

📚 Bonus: Function Expression

You can also store a function in a variable:


const square = function(num) {
  return num * num;
};

console.log(square(4));  // 16
