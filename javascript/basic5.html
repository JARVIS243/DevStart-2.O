JavaScript: Lesson 5 – Input & Output

How to get input from users and show output to the screen or console

🔎 What is Input & Output?

In any program, you need to:
👉 Take input from the user 👉 Show output to the user
JavaScript provides easy ways to do both.

📥 Input Using prompt()

The prompt() function shows a text box to get user input:


let name = prompt("What is your name?");
console.log(name); // shows what user typed

This creates a pop-up asking for input. The value entered is stored in the variable.

📤 Output Methods

  • alert() – Show a pop-up message
  • console.log() – Print to browser console (for developers)
  • document.write() – Write directly to the webpage

alert("Hello, world!");
console.log("This is from DevStart");
document.write("Welcome to JavaScript!");

🧠 Important Notes

  • prompt() always returns a string
  • To do math, you may need to convert input to a number

let age = prompt("Enter your age:");
let ageNumber = Number(age);  // convert string to number
console.log(ageNumber + 1);   // adds 1

🧪 Try This

  • Ask the user for their name and greet them using alert()
  • Ask for two numbers using prompt() and show their sum
  • Print the final result using console.log()

📚 Bonus: Why Use Console?

The browser console is a powerful tool for testing and debugging. Right-click on any webpage → Inspect → Console tab
