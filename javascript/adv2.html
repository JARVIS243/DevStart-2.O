JavaScript: Advanced Lesson 2 – Arrays

Store and manage multiple values in a single variable using arrays

🧺 What is an Array?

An array is a list that stores multiple values in one variable. You can access each item by its position (index).

📦 Creating an Array


let fruits = ["apple", "banana", "cherry"];
console.log(fruits);  // ["apple", "banana", "cherry"]

Values inside an array are called elements. They can be strings, numbers, booleans, or even other arrays!

🔍 Accessing Elements


console.log(fruits[0]); // "apple"
console.log(fruits[1]); // "banana"

Arrays start counting from 0. So the first element is [0], second is [1], etc.

🔁 Looping Through an Array


for (let i = 0; i < fruits.length; i++) {
  console.log(fruits[i]);
}

fruits.length gives the total number of items in the array.

➕ Adding and Removing Items


fruits.push("mango");     // Add to end
fruits.pop();             // Remove last
fruits.unshift("kiwi");   // Add to start
fruits.shift();           // Remove first

📌 Common Array Methods

  • push() – add to end
  • pop() – remove from end
  • unshift() – add to start
  • shift() – remove from start
  • length – number of items
  • indexOf(value) – find position
  • includes(value) – check existence

console.log(fruits.includes("banana")); // true
console.log(fruits.indexOf("cherry"));  // 2

📚 Bonus: Array of Mixed Types

Arrays can hold any data type:


let mixed = [42, "hello", true, null];
console.log(mixed[1]); // "hello"
