JavaScript: Lesson 1 – Introduction

Understand what JavaScript is and why it matters in web development

🌐 What is JavaScript?

JavaScript is a programming language that brings life to websites. It's what allows buttons to work, forms to react, and pages to update without reloading. While HTML gives the structure and CSS the style — JavaScript adds interactivity and logic.

🧠 Why Learn JavaScript First?

  • ✅ Runs in every browser (no need to install)
  • ✅ Easy to start with – no setup required
  • ✅ Used for both frontend and backend (Node.js)
  • ✅ Great for logic development with instant results

💻 First Program – "Hello, World"


alert("Hello, world!");

This creates a pop-up message in your browser.

🎯 Let's Understand It Step-by-Step

  • alert() – A built-in JavaScript function
  • "Hello, world!" – A string (text) you are showing
  • Function call – You're telling the browser to "do something"

This is your first example of logic: call a function to perform an action!

🧪 Try This

  • Replace "Hello, world!" with your name
  • Try calling console.log("This is from DevStart") in browser console
  • Use window.alert("Welcome!") – same as alert()

📚 Bonus: Where Does JavaScript Run?

JavaScript runs directly inside the browser (like Chrome, Firefox, Edge). You can also run it on the backend using Node.js, which makes JavaScript even more powerful.
