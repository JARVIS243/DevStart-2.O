JavaScript: Lesson 3 – Data Types

Understand the different types of values your program can handle

📦 What is a Data Type?

Every value in JavaScript has a type — it tells the computer what kind of data it is. Knowing the data type helps JavaScript decide how to use and process the value.

🔢 Common Data Types

  • String: Text — anything inside quotes
  • Number: Any number (integer or decimal)
  • Boolean: true or false (yes/no logic)
  • Undefined: A variable declared but not assigned
  • Null: Intentional empty value

💡 Examples


let name = "Aju";         // String
let age = 18;             // Number
let isStudent = true;     // Boolean
let phone;                // Undefined
let marks = null;         // Null

Each line uses a different data type. You’ll work with these types in every program.

🎯 Checking Data Types


let name = "DevStart";
console.log(typeof name);  // Output: string

let age = 20;
console.log(typeof age);   // Output: number

Use typeof to check what type of data is inside a variable.

🧪 Try This

  • Create variables for email, height, and isLoggedIn
  • Use console.log(typeof ...) to check their types
  • Try changing values and see how the type stays the same or changes

📚 Bonus: Dynamic Typing

JavaScript is dynamically typed – a variable’s type can change: 


let data = "hello";  // string
data = 42;           // now number
console.log(typeof data); // number

This gives flexibility, but be careful to track your types!
