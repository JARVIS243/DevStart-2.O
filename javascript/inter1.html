JavaScript: Intermediate Lesson 1 – Conditionals

Make decisions in your code using if, else, and else if

🔍 What is a Conditional Statement?

A conditional allows your program to make decisions. It checks if something is true and runs different code based on that.

✅ Syntax of if


let age = 18;

if (age >= 18) {
  console.log("You can vote.");
}

If the condition is true, the code inside the block runs.

🛑 Add else for the opposite case


let age = 16;

if (age >= 18) {
  console.log("You can vote.");
} else {
  console.log("You are too young to vote.");
}

🔁 Use else if for multiple options


let marks = 70;

if (marks >= 90) {
  console.log("Grade: A");
} else if (marks >= 75) {
  console.log("Grade: B");
} else if (marks >= 50) {
  console.log("Grade: C");
} else {
  console.log("Fail");
}

⚠️ Common Mistakes to Avoid

  • Always use curly braces { } to group blocks
  • Use == to compare values, = is for assignment
  • Conditions must return true or false

🧪 Try This

  • Ask the user for their age using prompt()
  • Use if-else to print if they’re a minor or adult
  • Try checking if a number is even or odd using %

📚 Bonus: Truthy and Falsy Values

Some values behave like true/false in conditions: 


if ("hello") {
  console.log("This will run");
}

if (0) {
  console.log("This won't run");  // 0 is falsy
}

Falsy values: 0, "", null, undefined, false, NaN
