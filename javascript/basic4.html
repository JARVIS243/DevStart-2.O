JavaScript: Lesson 4 – Operators

Use symbols to perform actions like math, comparison, and logic

➕ What are Operators?

Operators are special symbols that tell JavaScript to do something with values. Just like math symbols, they combine or compare values in your code.

🔢 1. Arithmetic Operators

Used for calculations:


let a = 10, b = 3;

console.log(a + b); // ➕ 13
console.log(a - b); // ➖ 7
console.log(a * b); // ✖️ 30
console.log(a / b); // ➗ 3.33
console.log(a % b); // ➗ remainder = 1

🟰 2. Assignment Operators

Used to assign values to variables:


let x = 5;     // Assign
x += 3;        // Same as x = x + 3 (now 8)
x *= 2;        // x = x * 2 (now 16)

🤔 3. Comparison Operators

Used to compare values (result is true or false):


let a = 5, b = 8;

console.log(a > b);    // false
console.log(a < b);    // true
console.log(a == b);   // false (equal in value)
console.log(a != b);   // true (not equal)

Note: == checks value only. === checks value and type.

🔗 4. Logical Operators

Used to combine conditions (booleans):


let isAdult = true;
let hasID = false;

console.log(isAdult && hasID);  // AND – false
console.log(isAdult || hasID);  // OR – true
console.log(!isAdult);          // NOT – false

🧪 Try This

  • Try 10 > 5 && 4 < 2
  • Create two variables and compare them using == and ===
  • Practice math using % (modulus) to find remainders

📚 Bonus: Difference Between == and ===


let a = "5";     // string
let b = 5;       // number

console.log(a == b);  // true (value same)
console.log(a === b); // false (type is different)

Always prefer === to avoid confusion in larger programs.
