JavaScript: Lesson 2 – Variables

Store, update, and manage values using variables

🧠 What is a Variable?

A variable is a container that holds information (like numbers, text, etc.) that your program can use and update. Just like boxes with labels, variables store values with names.

📦 Creating Variables


let name = "Aju";
let age = 20;
let isStudent = true;

- let is used to declare a variable - "Aju" is a string (text) - 20 is a number - true is a boolean (yes/no)

🔁 Changing Variable Values


let score = 10;
score = 15;  // value updated
console.log(score);  // prints 15

You can update variables later in your program.

🧪 Try This

  • Create a variable called city and assign your hometown
  • Change its value to another city later
  • Print both values using console.log()

⚠️ What About var and const?

  • var is older, avoid using it now
  • let is modern and safe for changing values
  • const is used when the value shouldn't change

const pi = 3.14;
pi = 3.14159;  // ❌ Error: const can't be changed

📚 Bonus Tips

  • Variable names are case-sensitive (Name ≠ name)
  • Use clear names like userAge, totalMarks
  • Don't use spaces or special characters
