Python: Intermediate Lesson 3 – Functions

Create reusable blocks of logic using functions

🧠 What is a Function?

A function is a block of code that performs a specific task. You define it once, and call (use) it anywhere in your program.

🔧 Creating a Function


def greet():
    print("Hello, welcome to DevStart!")

greet()  # calling the function

🧾 Function with Parameters


def greet(name):
    print(f"Hello, {name}!")

greet("Aju")
greet("DevStart Student")

name is a parameter – a value passed to the function

🔙 Return Values


def add(a, b):
    return a + b

result = add(5, 3)
print("Sum is", result)

return sends the result back to where it was called

📚 Multiple Parameters


def multiply(x, y):
    return x * y

print(multiply(4, 5))  # 20

📌 Why Use Functions?

  • Reusability – write once, use many times
  • Clean code – break down big problems into smaller steps
  • Debug easily – test each function separately

🧪 Try This

  • Create a function that calculates area of a circle
  • Write a function to check if a number is even or odd
  • Create a function that takes 3 numbers and returns the largest

📌 Summary

  • def function_name(): defines a function
  • parameters pass input to functions
  • return sends output back
  • Functions keep code organized and reusable
