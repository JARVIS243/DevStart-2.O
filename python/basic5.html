Python: Lesson 5 – Conditional Statements

Make decisions in code using if, elif, and else

❓ Why Use Conditions?

Conditions allow your program to make decisions and take different actions based on certain situations.

✅ Syntax of if Statement


age = 18

if age >= 18:
    print("You are an adult")

- The condition age >= 18 is checked
- If it’s True, the indented block is executed

🔀 if...else


marks = 45

if marks >= 50:
    print("Pass")
else:
    print("Fail")

If the if condition is False, the else block runs.

📊 if...elif...else – Multiple Conditions


score = 85

if score >= 90:
    print("Grade A")
elif score >= 75:
    print("Grade B")
elif score >= 60:
    print("Grade C")
else:
    print("Needs Improvement")

The first matching condition runs, others are skipped.

⚠️ Indentation is Important

Python uses spaces (usually 4) to define blocks of code. Don’t forget to indent properly!

🧪 Try This

  • Ask the user for their age, and check if they are eligible to vote (age ≥ 18)
  • Check if a number is positive, negative, or zero using if/elif/else
  • Grade system: A (90+), B (75+), C (60+), D (below 60)

📌 Summary

  • if runs a block if condition is True
  • else runs if no conditions are True
  • elif checks other possibilities
  • Indentation is required for blocks
