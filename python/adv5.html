Python: Advanced Lesson 5 – Inheritance & Polymorphism

Build flexible, reusable code using object-oriented principles

🔗 What is Inheritance?

Inheritance allows one class (child) to use the properties and methods of another class (parent). This helps reduce code duplication and improve structure.

👪 Basic Inheritance Example


class Animal:
    def speak(self):
        print("The animal makes a sound")

class Dog(Animal):
    def bark(self):
        print("The dog barks")

d = Dog()
d.speak()  # Inherited
d.bark()   # Own method

🔁 Overriding Methods

A child class can change (override) a method from the parent class.


class Animal:
    def speak(self):
        print("The animal speaks")

class Cat(Animal):
    def speak(self):
        print("The cat meows")

c = Cat()
c.speak()  # Output: The cat meows

👨‍👦 Calling Parent Methods with super()


class Parent:
    def show(self):
        print("Parent show")

class Child(Parent):
    def show(self):
        super().show()
        print("Child show")

obj = Child()
obj.show()

🧠 What is Polymorphism?

Polymorphism means "many forms". It allows different classes to define the same method name with different behavior. 


class Bird:
    def sound(self):
        print("Bird chirps")

class Duck(Bird):
    def sound(self):
        print("Duck quacks")

def make_sound(animal):
    animal.sound()

make_sound(Bird())
make_sound(Duck())

🧪 Try This

  • Create a base class Vehicle with a method drive()
  • Create subclasses Car and Bike that override drive()
  • Call the method using a function that accepts any vehicle

📌 Summary

  • Inheritance = reuse code from parent class
  • Overriding = replace a parent method
  • super() = call parent method
  • Polymorphism = same method, different behavior
