Python: Advanced Lesson 3 – Modules & Packages

Break your code into organized, reusable pieces

📦 What is a Module?

A module is a Python file (.py) that contains variables, functions, or classes. You can use it in other files using import. 


# math_utils.py
def add(a, b):
    return a + b
      

# main.py
import math_utils
print(math_utils.add(5, 3))  # Output: 8

🧮 Built-in Modules


import math
print(math.sqrt(25))  # 5.0

import random
print(random.randint(1, 10))  # random number between 1–10

🧠 Importing Techniques


from math import sqrt
print(sqrt(16))  # 4.0

import math as m
print(m.pi)  # 3.14159...

📁 What is a Package?

A package is a folder that contains multiple modules (Python files) and an optional __init__.py file to mark it as a package. 


my_package/
│
├── __init__.py
├── module1.py
└── module2.py
      

from my_package import module1
module1.function_name()

🔁 Reusability Example

Create a module named greet.py:


def welcome(name):
    print(f"Hello, {name}!")

Then use it in another file:


import greet
greet.welcome("Aju")

🧪 Try This

  • Create a module with basic math operations
  • Import and use it from a different file
  • Create a package with two modules (e.g., math & strings)

📌 Summary

  • Module = single Python file
  • Package = folder with multiple modules
  • Use import, from ... import, and as to use them
  • Helps keep code clean, structured, and reusable
