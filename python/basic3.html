Python: Lesson 3 – Input & Output

Learn how to take input from users and show output in Python

📤 What is Output?

Output means showing information to the user. In Python, we use the print() function. 


print("Welcome to DevStart!")
print(10 + 5)

This will display messages and results on the screen.

📥 What is Input?

Input means getting data from the user using the input() function. 


name = input("Enter your name: ")
print("Hello, " + name + "!")

input() always returns a string. If you want a number, use int() or float().

🔢 Convert Input to Numbers


age = input("Enter your age: ")
print(age + 1)  # ❌ Error: str + int

# Correct way:
age = int(input("Enter your age: "))
print(age + 1)  # ✅ Adds 1

Use int() for whole numbers and float() for decimals.

🧪 Try This

  • Ask the user for their favorite number
  • Convert it to int and add 10
  • Print the result using print()

💡 Tip: Use f-strings for clean output


name = input("Your name: ")
age = int(input("Your age: "))

print(f"Hi {name}, next year you’ll be {age + 1} years old!")

F-strings are a modern and easy way to format output.
