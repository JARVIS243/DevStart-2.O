Python: Intermediate Lesson 1 – Loops

Repeat tasks using for and while loops

🔁 Why Use Loops?

Loops let you repeat code without writing it multiple times. Great for automation, counting, processing lists, and logic building.

1️⃣ for Loop – Loop with a Range


# Print numbers from 1 to 5
for i in range(1, 6):
    print(i)

range(start, stop) creates numbers from start to stop-1. i changes in each loop.

2️⃣ for Loop – Looping Through a List


fruits = ["apple", "banana", "mango"]

for fruit in fruits:
    print(fruit)

Each item in the list is printed one by one.

3️⃣ while Loop – Loop Until a Condition is False


count = 1

while count <= 5:
    print("Count is", count)
    count += 1

Be careful with while – if the condition never becomes False, the loop runs forever.

🔚 Breaking or Skipping Loops


for i in range(1, 6):
    if i == 3:
        continue  # skips 3
    if i == 5:
        break     # stops loop
    print(i)

continue skips current turn
break ends the loop early

🧪 Try This

  • Print even numbers from 1 to 20
  • Print your name 5 times using a loop
  • Make a countdown from 10 to 1
  • Try a loop that asks for input until user types "exit"

📌 Summary

  • for loop → known repetitions (e.g. range, list)
  • while loop → unknown repetitions (e.g. input until valid)
  • break → exit early
  • continue → skip current iteration
