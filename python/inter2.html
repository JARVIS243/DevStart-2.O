Python: Intermediate Lesson 2 – Lists

Store, access, and modify multiple values using lists

📦 What is a List?

A list is a collection of values stored in one variable. Lists are ordered, changeable, and allow duplicate values. 


fruits = ["apple", "banana", "mango"]
print(fruits)

🔢 Accessing List Items


print(fruits[0])  # apple
print(fruits[1])  # banana

Index starts from 0

✏️ Modifying List Items


fruits[1] = "orange"
print(fruits)  # ['apple', 'orange', 'mango']

📏 List Length


print(len(fruits))  # 3

➕ Adding Items


fruits.append("grape")        # Add to end
fruits.insert(1, "kiwi")      # Insert at index 1

➖ Removing Items


fruits.remove("apple")        # Remove by value
fruits.pop()                  # Remove last item
del fruits[0]                 # Delete by index

🔁 Loop Through a List


for fruit in fruits:
    print(fruit)

🔄 Other Useful List Methods


fruits.sort()       # Sort alphabetically
fruits.reverse()    # Reverse order
fruits.clear()      # Remove all items

🧪 Try This

  • Create a list of your 5 favorite movies
  • Print the 2nd and 4th movies
  • Replace the last movie with a new one
  • Loop through and print all movies
  • Sort the list in alphabetical order

📌 Summary

  • Lists are ordered, changeable, and indexed
  • Use append(), insert(), remove() to modify them
  • Loop using for
  • Check size with len()
