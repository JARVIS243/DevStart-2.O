Python: Lesson 2 – Variables & Data Types

Store and work with different kinds of information using variables

📦 What is a Variable?

A variable is a name used to store data in memory. Think of it as a container that holds values. 


name = "Aju"
age = 18

Now name holds a string, and age holds a number.

🧠 Variable Naming Rules

  • Use letters, numbers, and underscores (_)
  • Must start with a letter or underscore (not a number)
  • No spaces or special characters
  • Use lowercase_with_underscores for readability

🧪 Examples


username = "devstart"
score = 95
pi = 3.14
is_student = True

Each variable stores a value of a specific data type.

🔤 Data Types in Python

  • str – Text (e.g. "Hello")
  • int – Whole numbers (e.g. 10)
  • float – Decimal numbers (e.g. 3.14)
  • bool – True or False

print(type("Hello"))   # str
print(type(42))        # int
print(type(3.5))       # float
print(type(True))      # bool

🧪 Try This

  • Create variables for your name, age, and favorite number
  • Print their types using type()
  • Try mixing int and str and see what happens

⚠️ Bonus: Strings + Numbers


name = "Aju"
age = 18
print(name + age)   # ❌ Error: can't mix str and int
print(name + str(age))  # ✅ Works

To join a number with a string, you must first convert it using str().
