Python: Advanced Lesson 1 – File Handling

Read from and write to files using Python

📄 Why File Handling?

File handling allows your program to read, write, and manage external files like text files. It's used for saving data permanently (like user info, logs, etc.).

📖 Reading a File


# Make sure "data.txt" exists in your folder
file = open("data.txt", "r")
content = file.read()
print(content)
file.close()

✍️ Writing to a File


file = open("data.txt", "w")
file.write("Welcome to DevStart!\n")
file.write("This is Python file handling.")
file.close()

➕ Appending to a File


file = open("data.txt", "a")
file.write("\nThis line is added later.")
file.close()

✅ Reading Line by Line


file = open("data.txt", "r")
for line in file:
    print(line.strip())
file.close()

💡 Using with Keyword (Best Practice)


with open("data.txt", "r") as file:
    print(file.read())

🧪 Try This

  • Create a file and write a list of your favorite languages
  • Read the file and print each language
  • Append a new language to the file
  • Use with instead of open/close

📌 Summary

  • open("file", mode) – open file
  • r = read, w = write, a = append
  • close() – always close after use
  • with – auto closes file
