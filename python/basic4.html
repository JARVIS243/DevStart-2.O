Python: Lesson 4 – Operators

Perform calculations and comparisons in Python using operators

➕ What are Operators?

Operators are symbols used to perform actions like addition, subtraction, comparison, etc.

🔢 Arithmetic Operators


a = 10
b = 3

print(a + b)  # Addition → 13
print(a - b)  # Subtraction → 7
print(a * b)  # Multiplication → 30
print(a / b)  # Division → 3.333...
print(a // b) # Floor Division → 3
print(a % b)  # Modulus (remainder) → 1
print(a ** b) # Exponent → 1000

⚖️ Comparison Operators

Used to compare two values. Returns True or False.


x = 5
y = 10

print(x == y)   # False
print(x != y)   # True
print(x > y)    # False
print(x < y)    # True
print(x >= y)   # False
print(x <= y)   # True

🔁 Assignment Operators

Used to assign or update values in variables.


x = 5
x += 2   # same as x = x + 2 → 7
x *= 3   # x = x * 3 → 21
x -= 1   # x = x - 1 → 20

🔍 Logical Operators

Used to combine multiple conditions.


age = 20

print(age > 18 and age < 25)  # True
print(age > 25 or age == 20)  # True
print(not age == 18)          # True

  • Write a program to check if a number is even or odd using modulus %
  • Compare two numbers and print which one is larger
  • Try using and / or / not with conditions

📌 Summary

  • Arithmetic → + - * / // % **
  • Comparison → == != > < >= <=
  • Logical → and or not
  • Assignment → = += -= *=
