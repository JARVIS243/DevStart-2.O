Python: Intermediate Lesson 5 – Dictionaries

Store and access data using key-value pairs

🔑 What is a Dictionary?

A dictionary is a collection of key-value pairs. Each key is unique and maps to a specific value. 


student = {
    "name": "Aju",
    "age": 20,
    "grade": "A"
}

🔍 Access Values


print(student["name"])   # Aju
print(student["grade"])  # A

✏️ Modify & Add Entries


student["age"] = 21
student["city"] = "Kochi"

Update existing values or add new key-value pairs.

➖ Remove Items


del student["grade"]  # Removes grade

🔁 Loop Through Dictionary


for key, value in student.items():
    print(key, ":", value)

🔧 Useful Methods


print(student.keys())    # all keys
print(student.values())  # all values
print(student.get("age"))   # 21
print(student.get("marks", "Not Found"))  # fallback

📦 Nested Dictionary


students = {
    "aju": {"age": 20, "grade": "A"},
    "arun": {"age": 19, "grade": "B"}
}
print(students["aju"]["grade"])  # A

🧪 Try This

  • Create a dictionary with your name, age, and course
  • Update your course to "Python Mastery"
  • Loop through and print all details
  • Create a nested dictionary of 2 students

📌 Summary

  • Dictionaries use keys to access values
  • Use { } to define and [] to access
  • Supports looping, updating, deleting, and nesting
