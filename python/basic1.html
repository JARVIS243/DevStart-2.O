Python: Lesson 1 – Introduction

A beginner-friendly introduction to Python programming

🐍 What is Python?

Python is a high-level, easy-to-read programming language used in:

  • Web Development
  • Data Science
  • Machine Learning & AI
  • Automation and Scripting
  • Game Development

💡 Why Learn Python First?

  • Simple syntax (similar to English)
  • Popular – used by Google, YouTube, Instagram, NASA
  • Strong community & libraries
  • Perfect for beginners

🔧 How to Run Python Code

  • Use online tools: Replit, Programiz
  • Or install Python from python.org

✅ Your First Python Code


print("Hello, world!")

This prints a message to the screen. It's your first program!

📚 Python File Extension

Save Python programs with .py extension. Example: hello.py

🧪 Try This

  • Write your name using print()
  • Print 3 different messages
  • Try printing numbers too: print(123)

📌 Summary

  • Python is beginner-friendly and powerful
  • print() is used to show output
  • Practice makes perfect – start writing now!
