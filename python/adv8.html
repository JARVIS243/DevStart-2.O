Python: Advanced Lesson 8 – Python Libraries Overview

Explore the most widely used Python libraries for different fields

📚 What is a Library?

A library is a collection of pre-written Python code that helps you do complex tasks easily. You install them using pip and import them into your code.

🔢 NumPy – Numerical Python


import numpy as np

arr = np.array([1, 2, 3])
print(arr.mean())  # Output: 2.0

✅ Used for arrays, matrices, scientific computing

📊 pandas – Data Analysis


import pandas as pd

data = {"name": ["Aju", "Devi"], "age": [18, 21]}
df = pd.DataFrame(data)
print(df)

✅ Create tables, filter data, import CSV/Excel

📈 matplotlib – Data Visualization


import matplotlib.pyplot as plt

plt.plot([1, 2, 3], [4, 5, 6])
plt.title("Line Graph")
plt.show()

✅ Draw graphs and charts (line, bar, pie...)
