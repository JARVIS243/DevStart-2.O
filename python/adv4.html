Python: Advanced Lesson 4 – Classes & Objects

Use Object-Oriented Programming (OOP) to organize and reuse code efficiently

🏗️ What is OOP?

OOP (Object-Oriented Programming) is a programming style where we build programs using objects and classes. It improves reusability, structure, and scalability.

📦 What is a Class?

A class is a blueprint for creating objects. It defines properties (variables) and behaviors (functions). 


class Student:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def greet(self):
        print("Hello, my name is", self.name)

👤 What is an Object?

An object is an instance of a class. It has its own data and functions. 


s1 = Student("Aju", 18)
s1.greet()   # Output: Hello, my name is Aju

⚙️ __init__ Method

The __init__ method is called automatically when an object is created. It's used to initialize values.

🏷️ Object Attributes


print(s1.name)  # Aju
print(s1.age)   # 18

📌 More Methods


class Car:
    def __init__(self, brand):
        self.brand = brand

    def drive(self):
        print(self.brand, "is driving")

c = Car("Tesla")
c.drive()
