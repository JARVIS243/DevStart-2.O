Python: Intermediate Lesson 4 – Strings

Work with text using Python’s powerful string features

🔤 What is a String?

A string is a sequence of characters, used to store text data. 


name = "DevStart"
message = 'Learn Python!'

➕ Joining Strings


first = "Hello"
second = "World"
print(first + " " + second)  # Hello World

📏 String Length


text = "DevStart"
print(len(text))  # 8

🔁 Looping Through a String


for letter in "Python":
    print(letter)

🔍 Useful String Methods


text = " Hello Python "

print(text.lower())     # hello python
print(text.upper())     # HELLO PYTHON
print(text.strip())     # Removes spaces around
print(text.replace("Python", "World"))  # Hello World
print(text.split())     # ['Hello', 'Python']

📘 More String Tricks


name = "aju"
print(name.capitalize())  # Aju
print(name.startswith("a"))  # True
print(name.endswith("u"))    # True

🔢 Accessing Characters


text = "Python"
print(text[0])     # P
print(text[-1])    # n

🧪 Try This

  • Ask the user to enter their name and greet them with capitalized name
  • Count the number of letters in a word
  • Replace one word in a sentence using replace()
  • Check if a string starts or ends with a specific letter

📌 Summary

  • Strings store text – use quotes
  • Use len(), lower(), strip(), replace(), etc.
  • Strings are sequences – you can loop through or access by index
