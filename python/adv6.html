Python: Advanced Lesson 6 – Working with JSON

Use JSON to store and exchange structured data in Python

📄 What is JSON?

JSON (JavaScript Object Notation) is a lightweight format for storing and transporting data. It's used widely in web APIs and configuration files.

📥 Importing the json Module


import json

🔁 Converting Python → JSON (Serialization)


data = {"name": "Aju", "age": 18, "language": "Python"}

json_string = json.dumps(data)
print(json_string)

Output: {"name": "Aju", "age": 18, "language": "Python"}

🔄 Converting JSON → Python (Deserialization)


json_data = '{"name": "Aju", "age": 18}'
parsed = json.loads(json_data)
print(parsed["name"])  # Aju

📂 Working with JSON Files


# Writing to a file
data = {"name": "DevStart", "type": "Platform"}
with open("data.json", "w") as file:
    json.dump(data, file)

# Reading from a file
with open("data.json", "r") as file:
    content = json.load(file)
    print(content)

🧪 Try This

  • Create a list of students as dictionaries
  • Save it to a file in JSON format
  • Read the file and display all names

📌 Summary

  • json.dumps() – Python to JSON (string)
  • json.loads() – JSON (string) to Python
  • json.dump() – Save Python data to file
  • json.load() – Load JSON from file
