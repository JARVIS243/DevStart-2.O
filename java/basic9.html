Java: Basic Lesson 9 – String Handling

Work with text using Java Strings

🧵 What is a String?

A String is a sequence of characters enclosed in double quotes. Java provides many useful methods to handle strings.

🧠 Declaring Strings


String name = "Aju";
String greeting = "Welcome to DevStart!";
System.out.println(name);
System.out.println(greeting);

📚 Common String Methods


String message = "  Java Programming  ";

System.out.println(message.length());        // 20
System.out.println(message.toUpperCase());  // "  JAVA PROGRAMMING  "
System.out.println(message.trim());         // "Java Programming"
System.out.println(message.contains("Java")); // true
System.out.println(message.indexOf("g"));   // 10

🔗 String Concatenation


String first = "Dev";
String second = "Start";

String full = first + second;
System.out.println(full);  // DevStart

📌 String Comparison


String a = "hello";
String b = "hello";

System.out.println(a.equals(b));     // true
System.out.println(a == b);          // true (in some cases)

String c = new String("hello");
System.out.println(a.equals(c));     // true
System.out.println(a == c);          // false (different objects)

🧪 Try This

  • Take a name input from user, convert to uppercase, and print
  • Check if a string contains the word "Java"
  • Join two user-given words into one string

📌 Summary

  • String is used for text
  • Use + to join strings
  • Use .equals() for string comparison
  • Helpful methods: length(), toUpperCase(), trim(), contains()
