Java: Advanced Lesson 1 – Exception Handling

Catch and handle runtime errors using try-catch blocks

💥 What is an Exception?

An exception is an event that disrupts the normal flow of a program. For example: dividing by zero, accessing invalid array index, file not found, etc.

🧯 Why Use Exception Handling?

  • To prevent program crash
  • To show user-friendly error messages
  • To handle unexpected conditions

📘 Example: try-catch


public class Main {
    public static void main(String[] args) {
        try {
            int a = 10 / 0;  // division by zero
            System.out.println(a);
        } catch (ArithmeticException e) {
            System.out.println("Error: Cannot divide by zero!");
        }

        System.out.println("Program continues...");
    }
}

🚨 Common Exception Types

  • ArithmeticException – divide by 0
  • ArrayIndexOutOfBoundsException
  • NullPointerException
  • FileNotFoundException

🧪 Try This

  • Write code to divide two numbers, catch division by 0
  • Access an invalid array index and handle it

📌 Summary

  • try – code that may throw an exception
  • catch – handles the exception
  • Prevent crashes and keep your app stable
