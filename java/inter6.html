Java: Intermediate Lesson 6 – Static Keyword

Use class-level variables and methods without creating objects

📘 What is static?

The static keyword is used to create variables and methods that belong to the class, rather than to specific objects. It allows shared values across all instances.

🔧 Static Variable Example


public class Student {
    static String school = "DevStart High";

    String name;

    Student(String n) {
        name = n;
    }

    void show() {
        System.out.println(name + " studies at " + school);
    }
}

public class Main {
    public static void main(String[] args) {
        Student s1 = new Student("Aju");
        Student s2 = new Student("Ravi");

        s1.show();
        s2.show();
    }
}

⚙️ Static Method


public class Utils {
    static void greet() {
        System.out.println("Welcome to DevStart!");
    }

    public static void main(String[] args) {
        Utils.greet();  // No need to create object
    }
}

📌 Key Points

  • static variables are shared across all objects
  • static methods can be called without creating objects
  • static blocks run once when class is loaded
  • static methods can’t access non-static members directly

🧪 Try This

  • Create a class MathTools with static method square()
  • Use it in main without creating an object
  • Make a static variable appName = "DevStart Java" and access it

📌 Summary

  • Static = shared and class-level
  • Use static methods when no object-specific data is needed
  • Helps save memory and organize utility code
