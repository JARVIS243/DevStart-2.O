A switch is useful when you have many if-else checks based on one variable. It improves readability when checking fixed values like numbers, days, menu options, etc.

    case value1:

    default:
        // Code if no case matches
}

ğŸ§  Example: Day of the Week


import java.util.Scanner;

public class Weekday {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);

        System.out.print("Enter day number (1-7): ");
        int day = input.nextInt();

        switch (day) {
            case 1:
                System.out.println("Sunday");
                break;
            case 2:
                System.out.println("Monday");
                break;
            case 3:
                System.out.println("Tuesday");
                break;
            case 4:
                System.out.println("Wednesday");
                break;
            case 5:
                System.out.println("Thursday");
                break;
            case 6:
                System.out.println("Friday");
                break;
            case 7:
                System.out.println("Saturday");
                break;
            default:
                System.out.println("Invalid day number");
        }

        input.close();
    }
}

ğŸ“Œ Important Notes

  • break stops the code from falling into next case
  • default is optional but good to include
  • Switch works best with int, char, String
