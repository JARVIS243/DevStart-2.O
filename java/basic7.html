Java: Basic Lesson 7 – Loops

Repeat actions using while, for, and do-while loops

🔁 Why Use Loops?

Loops are used to repeat a block of code multiple times. This helps avoid writing the same code again and again.

🧠 Types of Loops in Java

  • while – runs as long as condition is true
  • for – used when number of repetitions is known
  • do-while – runs at least once, then checks condition

🔄 Example: while loop


int i = 1;

while (i <= 5) {
    System.out.println("Hello " + i);
    i++;
}

🔁 Example: for loop


for (int i = 1; i <= 5; i++) {
    System.out.println("Number: " + i);
}

🔂 Example: do-while loop


int x = 1;

do {
    System.out.println("Printed at least once");
    x++;
} while (x < 1);

🧪 Try This

  • Print all even numbers from 1 to 20 using for loop
  • Use while loop to display a countdown from 10 to 1
  • Use do-while to ask a question until the answer is correct

📌 Summary

  • while – check then run
  • for – best for fixed repetitions
  • do-while – run once then check
  • Loops save time and reduce code repetition
