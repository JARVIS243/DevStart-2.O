The finally block in Java always executes after try and catch blocks, regardless of whether an exception was thrown or not. It's useful for cleanup code like closing files or releasing resources.

ğŸ“˜ Example


public class Main {
    public static void main(String[] args) {
        try {
            int result = 10 / 0;
        } catch (ArithmeticException e) {
            System.out.println("Exception caught: " + e.getMessage());
        } finally {
            System.out.println("Finally block always runs!");
        }
    }
}
