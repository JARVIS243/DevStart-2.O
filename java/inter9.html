An interface in Java is a completely abstract class used to define a set of methods that must be implemented by a class. It helps in achieving full abstraction and multiple inheritance.

    void makeSound();  // implicitly public and abstract
}

class Dog implements Animal {
    public void makeSound() {
        System.out.println("Dog barks");
    }
}
