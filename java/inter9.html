Java: Intermediate Lesson 9 – Interfaces

Define abstract behavior contracts that classes must follow

🔌 What is an Interface?

An interface in Java is a completely abstract class used to define a set of methods that must be implemented by a class. It helps in achieving full abstraction and multiple inheritance.

📘 Syntax Example


interface Animal {
    void makeSound();  // implicitly public and abstract
}

class Dog implements Animal {
    public void makeSound() {
        System.out.println("Dog barks");
    }
}

🚀 Usage


public class Main {
    public static void main(String[] args) {
        Dog d = new Dog();
        d.makeSound();  // Output: Dog barks
    }
}

🛠️ Key Rules

  • Use interface keyword
  • All methods are abstract by default
  • Use implements keyword in class
  • A class can implement multiple interfaces

📦 Multiple Interfaces


interface Printable {
    void print();
}

interface Showable {
    void show();
}

class Demo implements Printable, Showable {
    public void print() {
        System.out.println("Printing...");
    }

    public void show() {
        System.out.println("Showing...");
    }
}

🧪 Try This

  • Create an interface Playable with method play()
  • Implement it in Guitar and Piano classes
  • Call their play() methods

📌 Summary

  • Interface defines a contract that must be fulfilled
  • All methods are public and abstract by default
  • Supports multiple inheritance
  • Use implements to bind interface with class
