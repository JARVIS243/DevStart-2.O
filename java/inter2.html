Java: Intermediate Lesson 2 – Inheritance

Reuse properties and behaviors of one class in another

🧬 What is Inheritance?

Inheritance allows one class (child) to inherit properties and methods from another class (parent). This promotes code reuse and organizes logic better.

🔧 Basic Example


class Animal {
    void sound() {
        System.out.println("Animals make sounds");
    }
}

// Dog inherits from Animal
class Dog extends Animal {
    void bark() {
        System.out.println("Dog barks");
    }
}

public class Main {
    public static void main(String[] args) {
        Dog d = new Dog();
        d.sound(); // inherited
        d.bark();  // own method
    }
}

🔁 Types of Inheritance

  • Single Inheritance (one parent → one child)
  • Multilevel Inheritance (class A → B → C)
  • Hierarchical Inheritance (one parent → many children)
  • Note: Java does not support multiple inheritance with classes

🧪 Try This

  • Create a Vehicle class with method move()
  • Create a Car class that inherits Vehicle
  • Add a method fuelType() in Car

📌 Summary

  • extends is used to inherit from another class
  • Child class gets access to parent's methods and variables
  • Encourages reuse and organization
