Java: Basic Lesson 1 – Introduction & First Program

Get started with Java and write your first program

☕ What is Java?

Java is a popular, high-level, object-oriented programming language used to build desktop apps, web apps, Android apps, games, and more.
Write Once, Run Anywhere (WORA) – Java programs run on any platform using the Java Virtual Machine (JVM).

🖥️ Java Program Structure


// HelloWorld.java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, DevStart!");
    }
}
  • public class HelloWorld – Class name must match file name
  • main() – Entry point of the program
  • System.out.println() – Prints text to screen

⚙️ How to Run

  1. Save file as HelloWorld.java
  2. Open terminal in the file folder
  3. Compile: javac HelloWorld.java
  4. Run: java HelloWorld

🧪 Try This

  • Change the message to your name
  • Add another System.out.println() line

📌 Summary

  • Java is object-oriented and cross-platform
  • Every Java program has a main() method
  • Use System.out.println() to print output
  • Compile with javac and run with java
