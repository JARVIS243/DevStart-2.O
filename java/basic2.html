Java: Basic Lesson 2 – Variables & Data Types

Learn how to store data using variables in Java

📦 What is a Variable?

A variable is a container used to store data. In Java, you must specify the data type when creating a variable.

🔢 Common Data Types in Java

  • int – Whole numbers (e.g. 10, -5)
  • double – Decimal numbers (e.g. 3.14)
  • char – Single character (e.g. 'A')
  • String – Sequence of characters (e.g. "Hello")
  • boolean – true or false

🧠 Variable Examples


public class Variables {
    public static void main(String[] args) {
        int age = 18;
        double height = 5.9;
        char grade = 'A';
        String name = "Aju";
        boolean passed = true;

        System.out.println(name + " is " + age + " years old.");
        System.out.println("Height: " + height + " ft");
        System.out.println("Grade: " + grade);
        System.out.println("Passed: " + passed);
    }
}

🧪 Try This

  • Create variables for your name, age, and favorite subject
  • Print a sentence using all three

📌 Summary

  • Java is statically typed – you must define the variable type
  • Use + to join strings and variables in output
  • Common types: int, double, char, String, boolean
