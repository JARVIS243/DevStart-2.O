Java: Advanced Lesson 16 – Multithreading

Run multiple threads to achieve concurrent execution

🧵 What is Multithreading?

Multithreading lets you run multiple threads (lightweight processes) simultaneously within a program, improving performance.

📘 Creating Threads


// Using Thread class
class MyThread extends Thread {
    public void run() {
        System.out.println("Thread running: " + Thread.currentThread().getName());
    }
}

public class Main {
    public static void main(String[] args) {
        MyThread t1 = new MyThread();
        t1.start();
    }
}

📘 Using Runnable Interface


class MyRunnable implements Runnable {
    public void run() {
        System.out.println("Runnable thread: " + Thread.currentThread().getName());
    }
}

public class Main {
    public static void main(String[] args) {
        Thread t = new Thread(new MyRunnable());
        t.start();
    }
}

🧠 Thread Lifecycle

  • New → Runnable → Running → Terminated
  • Use methods: start(), sleep(), join()

📌 Synchronization

Synchronize shared resources to avoid race conditions.


synchronized void method() {
    // thread-safe code
}

🧪 Try This

  • Create two threads printing numbers 1-5
  • Use sleep() to pause between prints
  • Experiment with synchronized methods
