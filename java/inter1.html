Java: Intermediate Lesson 1 – Constructors

Initialize objects with default or custom values

🏗️ What is a Constructor?

A constructor is a special method that is automatically called when an object is created. It is used to initialize object values.

🧠 Default Constructor


public class Student {
    String name;

    // Default constructor
    Student() {
        name = "Unknown";
    }

    void display() {
        System.out.println("Name: " + name);
    }
}

🛠️ Parameterized Constructor


public class Student {
    String name;

    // Constructor with parameter
    Student(String n) {
        name = n;
    }

    void display() {
        System.out.println("Name: " + name);
    }
}

🚀 Using the Constructor


public class Main {
    public static void main(String[] args) {
        Student s1 = new Student("Aju");
        s1.display();  // Output: Name: Aju
    }
}

🧪 Try This

  • Create a Book class with title and author
  • Use constructor to assign values
  • Print book details using a method

📌 Summary

  • Constructor is used to initialize objects
  • Default constructor has no parameters
  • Parameterized constructor allows setting values
  • Constructor name must match the class name
