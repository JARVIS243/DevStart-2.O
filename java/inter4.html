ğŸ” What is Method Overriding?

Method Overriding means providing a new version of a method from the parent class inside the child class. It lets you change or extend the behavior of inherited methods.

ğŸ“˜ Example


class Animal {
    void sound() {
        System.out.println("Animal makes a sound");
    }
}

class Dog extends Animal {
    @Override
    void sound() {
        System.out.println("Dog barks");
    }
}

public class Main {
    public static void main(String[] args) {
        Animal a = new Dog();
        a.sound();  // Output: Dog barks
    }
}

  • Method name, return type, and parameters must match exactly
  • Use @Override annotation to indicate overriding
  • Overriding happens in runtime (dynamic binding)
