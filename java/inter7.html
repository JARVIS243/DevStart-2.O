Java: Intermediate Lesson 7 – Final Keyword

Restrict modification of variables, methods, and classes

🔒 What is final?

The final keyword is used to restrict the user from changing something. It can be applied to:

  • Variables: Value cannot be changed
  • Methods: Cannot be overridden
  • Classes: Cannot be inherited

📌 Final Variable


public class Example {
    final int number = 10;

    void show() {
        // number = 20; // ❌ Error: can't change final variable
        System.out.println("Number is " + number);
    }
}

🛑 Final Method


class A {
    final void show() {
        System.out.println("This cannot be overridden");
    }
}

class B extends A {
    // void show() { } // ❌ Error
}

🚫 Final Class


final class Car {
    void start() {
        System.out.println("Car starts");
    }
}

// class SportCar extends Car { } // ❌ Error: cannot inherit final class

🧪 Try This

  • Create a final variable and try changing it
  • Create a final method in a parent class and try to override it
  • Create a final class and try to extend it

📌 Summary

  • final variable → constant value
  • final method → cannot override
  • final class → cannot be inherited
  • Useful for securing code structure
