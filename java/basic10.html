Java: Basic Lesson 10 – Intro to Object-Oriented Programming

Understand the basics of classes and objects in Java

🎯 What is OOP?

Object-Oriented Programming (OOP) is a style of programming where we structure code using classes and objects.

  • Class: Blueprint/template (e.g. Car)
  • Object: Real entity created from class (e.g. myCar)

🧱 Defining a Class


public class Student {
    String name;
    int age;

    void greet() {
        System.out.println("Hello, my name is " + name);
    }
}

🚀 Creating an Object


public class Main {
    public static void main(String[] args) {
        Student s1 = new Student();
        s1.name = "Aju";
        s1.age = 18;
        s1.greet();
    }
}

This will print:
Hello, my name is Aju

💡 Why Use OOP?

  • Organizes code better
  • Reusability (using classes again)
  • Makes large programs easier to manage

🧪 Try This

  • Create a Car class with brand, model, and year
  • Create 2 car objects and print their details
  • Add a method that prints “Engine started!”

📌 Summary

  • OOP is based on classes and objects
  • Class = template, Object = real entity
  • Use new keyword to create objects
  • OOP helps with modularity and reusability
