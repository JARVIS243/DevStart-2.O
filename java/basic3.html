Java: Basic Lesson 3 – User Input & Output

Read user input and display output using Scanner

📥 Getting Input from User

Java uses the Scanner class to read user input from the keyboard. First, import it from java.util package.

✍️ Example: Take Name and Age from User


import java.util.Scanner;

public class UserInput {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);

        System.out.print("Enter your name: ");
        String name = input.nextLine();

        System.out.print("Enter your age: ");
        int age = input.nextInt();

        System.out.println("Welcome, " + name + "! You are " + age + " years old.");

        input.close();
    }
}

📘 Input Methods

  • nextLine() – Reads a whole line (text)
  • nextInt() – Reads an integer
  • nextDouble() – Reads a decimal
  • nextBoolean() – true/false input

🧪 Try This

  • Ask the user for name, subject, and marks
  • Print a message like: “Aju scored 95 in Maths.”

📌 Summary

  • Use Scanner to read user input
  • Import it with import java.util.Scanner;
  • Always close scanner using input.close();
