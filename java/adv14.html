Java: Advanced Lesson 14 – Networking Basics

Connect programs over a network using sockets

🌐 What is Networking in Java?

Java provides built-in libraries for networking using classes in java.net. You can create network applications like chat apps, web clients, and servers.

🔌 Key Networking Classes

  • Socket – used by clients to connect to servers
  • ServerSocket – used by servers to listen for client requests
  • InetAddress – represents an IP address

📘 Example: Server Program


import java.net.*;
import java.io.*;

public class Server {
    public static void main(String[] args) throws IOException {
        ServerSocket server = new ServerSocket(1234);
        System.out.println("Server is waiting...");

        Socket socket = server.accept();
        System.out.println("Client connected!");

        DataInputStream input = new DataInputStream(socket.getInputStream());
        String message = input.readUTF();
        System.out.println("Message: " + message);

        input.close();
        socket.close();
        server.close();
    }
}

📘 Example: Client Program


import java.net.*;
import java.io.*;

public class Client {
    public static void main(String[] args) throws IOException {
        Socket socket = new Socket("localhost", 1234);

        DataOutputStream output = new DataOutputStream(socket.getOutputStream());
        output.writeUTF("Hello from client!");

        output.close();
        socket.close();
    }
}

📌 Notes

  • Run the Server first
  • Then run the Client (they must be on the same machine or reachable over network)
  • Use try-catch for production-level apps to handle exceptions

🧪 Try This

  • Modify the server to reply back to the client
  • Use BufferedReader and PrintWriter instead of DataStream
  • Make a simple echo server

📌 Summary

  • Socket and ServerSocket allow two-way communication
  • Use streams to send/receive messages
  • Java makes it easy to build basic network applications
