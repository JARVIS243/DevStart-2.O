Java: Advanced Lesson 17 – JavaFX Basics

Build modern GUI applications in Java

🎨 What is JavaFX?

JavaFX is a modern GUI toolkit for Java, used to create desktop applications with rich user interfaces.

📘 Basic JavaFX Application


import javafx.application.Application;
import javafx.scene.Scene;
import javafx.scene.control.Button;
import javafx.scene.layout.StackPane;
import javafx.stage.Stage;

public class Main extends Application {
    @Override
    public void start(Stage primaryStage) {
        Button btn = new Button("Click Me");
        btn.setOnAction(e -> System.out.println("Button clicked!"));

        StackPane root = new StackPane();
        root.getChildren().add(btn);

        Scene scene = new Scene(root, 300, 200);
        primaryStage.setTitle("JavaFX Demo");
        primaryStage.setScene(scene);
        primaryStage.show();
    }

    public static void main(String[] args) {
        launch(args);
    }
}

🔧 Key JavaFX Components

  • Stage – main window
  • Scene – container for all content
  • Controls – buttons, text fields, labels, etc.
  • Layouts – organize controls (VBox, HBox, GridPane)
