Java: Advanced Lesson 12 – Streams API

Process collections of data in a functional style using Streams

🌊 What is Streams API?

The Streams API in Java lets you process data (like collections) using a pipeline of operations (filtering, mapping, reducing, etc.) in a clean and functional style.

📘 Basic Example


import java.util.*;
import java.util.stream.*;

public class Main {
    public static void main(String[] args) {
        List<String> names = Arrays.asList("Anna", "John", "Mike", "Amanda");

        names.stream()
            .filter(name -> name.startsWith("A"))
            .forEach(System.out::println);
    }
}

🔧 Common Stream Operations

  • filter() – filter elements based on a condition
  • map() – transform elements
  • forEach() – perform an action for each element
  • collect() – convert stream back to list/set
  • sorted() – sort elements
  • reduce() – reduce elements to a single value

📘 Example: Mapping and Reducing


import java.util.*;
import java.util.stream.*;

public class Main {
    public static void main(String[] args) {
        List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5);

        int sum = numbers.stream()
            .map(n -> n * 2)
            .reduce(0, Integer::sum);

        System.out.println("Sum of doubled numbers: " + sum);
    }
}

📌 Benefits

  • Write clean, readable, functional-style code
  • Chain multiple operations easily
  • Parallel stream support for performance
