Java: Basic Lesson 5 – If-Else Statements

Make decisions in your program using conditions

🔍 What is If-Else?

If-else allows your program to make decisions based on conditions. It helps you control which block of code should run.

✅ Syntax


if (condition) {
    // Code runs if condition is true
} else {
    // Code runs if condition is false
}

🧠 Example: Check Pass or Fail


import java.util.Scanner;

public class PassCheck {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);

        System.out.print("Enter your mark: ");
        int mark = input.nextInt();

        if (mark >= 50) {
            System.out.println("✅ You passed!");
        } else {
            System.out.println("❌ You failed.");
        }

        input.close();
    }
}

🔁 Else-If Ladder


int score = 85;

if (score >= 90) {
    System.out.println("Grade A");
} else if (score >= 75) {
    System.out.println("Grade B");
} else if (score >= 60) {
    System.out.println("Grade C");
} else {
    System.out.println("Fail");
}

🧪 Try This

  • Ask the user for their age
  • Print if they are eligible to vote (18+)
  • Use else-if to show different age group messages

📌 Summary

  • if: runs code if condition is true
  • else: runs if condition is false
  • else-if: checks multiple conditions in order
