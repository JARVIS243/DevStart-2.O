Java: Advanced Lesson 11 – Lambda Expressions

Write cleaner and more concise code using functional programming

⚡ What is a Lambda Expression?

A lambda expression is a short block of code that takes in parameters and returns a value. It's used primarily to implement functional interfaces in a more concise way.

📘 Syntax


(parameters) -> { body }

Example:


(a, b) -> a + b
() -> System.out.println("Hello World")
name -> name.length()

🧪 Functional Interface


@FunctionalInterface
interface MyFunction {
    void greet();
}

public class Main {
    public static void main(String[] args) {
        MyFunction f = () -> System.out.println("Hello from Lambda!");
        f.greet();
    }
}

🧠 Built-in Functional Interfaces (java.util.function)

  • Consumer<T> – takes input, returns nothing
  • Supplier<T> – takes nothing, returns value
  • Function<T, R> – takes input and returns output
  • Predicate<T> – takes input and returns boolean

📘 Example: Using Predicate


import java.util.function.Predicate;

public class Main {
    public static void main(String[] args) {
        Predicate<String> isLong = str -> str.length() > 5;
        System.out.println(isLong.test("DevStart"));  // true
    }
}

📌 Summary

  • Lambda = concise way to implement a method
  • Only works with functional interfaces (single method)
  • Makes code shorter and more readable
