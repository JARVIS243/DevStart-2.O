Java: Basic Lesson 8 – Arrays

Store and access multiple values using arrays

📦 What is an Array?

An array is a collection of similar data items stored in a single variable. Each element can be accessed using its index (starting from 0).

🧠 Example: Declaring and Using Arrays


public class ArrayExample {
    public static void main(String[] args) {
        int[] numbers = {10, 20, 30, 40, 50};

        System.out.println("First number: " + numbers[0]);
        System.out.println("Last number: " + numbers[4]);
    }
}

🔁 Loop Through Arrays


int[] marks = {80, 90, 85, 95, 100};

for (int i = 0; i < marks.length; i++) {
    System.out.println("Subject " + (i+1) + ": " + marks[i]);
}

🧪 Try This

  • Create an array of 5 names and print each using a loop
  • Find the total and average of an array of 5 marks
  • Change the value at index 2 and print it again

📌 Summary

  • int[] arr = new int[5]; – creates empty array
  • arr[i] – access item at index i
  • .length – gives total number of elements
  • Arrays are zero-indexed and fixed in size
