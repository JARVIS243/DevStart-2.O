Java: Advanced Lesson 13 – File and Directory Operations

Read, write, create, and delete files and folders in Java

📁 Working with Files and Folders

Java provides two main packages for file operations:

  • java.io – basic file reading/writing
  • java.nio.file – advanced file and path handling

📘 Writing to a File


import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        FileWriter writer = new FileWriter("output.txt");
        writer.write("Welcome to DevStart 2.0!\nJava File Handling Example.");
        writer.close();
        System.out.println("File written successfully.");
    }
}

📖 Reading from a File


import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader reader = new BufferedReader(new FileReader("output.txt"));
        String line;
        while ((line = reader.readLine()) != null) {
            System.out.println(line);
        }
        reader.close();
    }
}

📂 Creating and Deleting Files/Directories


import java.nio.file.*;
import java.io.IOException;

public class Main {
    public static void main(String[] args) throws IOException {
        Path path = Paths.get("newfile.txt");

        // Create file
        Files.createFile(path);
        System.out.println("File created: " + path);

        // Delete file
        Files.delete(path);
        System.out.println("File deleted: " + path);

        // Create directory
        Path dir = Paths.get("myfolder");
        Files.createDirectory(dir);
        System.out.println("Directory created: " + dir);
    }
}

🧪 Try This

  • Create a folder called logs
  • Write some text to logs/devstart.txt
  • Read the file and print contents
  • Delete the file afterward

📌 Summary

  • Use FileWriter and BufferedReader for basic text file operations
  • Use java.nio.file for file/directory creation and deletion
  • Always close your file streams
  • Wrap in try-catch or use throws IOException
