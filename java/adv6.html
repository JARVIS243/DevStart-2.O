Java: Advanced Lesson 6 – Multithreading

Run multiple tasks simultaneously using threads

🚀 What is Multithreading?

Multithreading is a Java feature that allows the concurrent execution of two or more threads (lightweight processes). It's useful for performing multiple tasks at the same time (e.g., animation, downloading, calculations).

📘 Creating a Thread (2 Ways)

1. By Extending Thread Class


class MyThread extends Thread {
    public void run() {
        System.out.println("Thread is running...");
    }
}

public class Main {
    public static void main(String[] args) {
        MyThread t = new MyThread();
        t.start();  // starts the thread
    }
}

2. By Implementing Runnable Interface


class MyRunnable implements Runnable {
    public void run() {
        System.out.println("Runnable thread running...");
    }
}

public class Main {
    public static void main(String[] args) {
        Thread t = new Thread(new MyRunnable());
        t.start();
    }
}

⏳ Thread Methods

  • start() – begins execution
  • run() – contains the code to execute
  • sleep(ms) – pauses for milliseconds
  • join() – waits for another thread to finish
  • isAlive() – checks if thread is active

🧪 Try This

  • Create two threads: one prints numbers, one prints letters
  • Run both simultaneously
  • Try using sleep() to control timing

📌 Summary

  • Multithreading = running multiple tasks at once
  • Two main ways: extend Thread or implement Runnable
  • Use start() to begin thread execution
  • Use sleep(), join() for control
