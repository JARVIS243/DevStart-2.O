ğŸ“¦ What Are Generics?

Generics allow you to define classes, methods, and interfaces with a placeholder for data types.

This helps you avoid code duplication and increases type safety.

ğŸ“„ Generic Method Example


class Utilities
{
    public void PrintItem<T>(T item)
    {
        Console.WriteLine("Item: " + item);
    }
}

// Usage
Utilities util = new Utilities();
util.PrintItem<int>(100);         // Output: Item: 100
util.PrintItem<string>("DevStart"); // Output: Item: DevStart

ğŸ“„ Generic Class Example


class Box<T>
{
    public T Value;
}

// Usage
Box<int> intBox = new Box<int>();
intBox.Value = 42;

Box<string> strBox = new Box<string>();
strBox.Value = "Hello";
