C# – Basic 5: Conditional Statements

Make decisions in your program using if, else, and switch

❓ Why Conditions?

Conditions allow your program to choose what to do based on values and logic.

🔹 if / else if / else


int number = 10;

if (number > 0)
{
    Console.WriteLine("Positive");
}
else if (number < 0)
{
    Console.WriteLine("Negative");
}
else
{
    Console.WriteLine("Zero");
}

🔸 switch Statement


int day = 2;

switch (day)
{
    case 1:
        Console.WriteLine("Monday");
        break;
    case 2:
        Console.WriteLine("Tuesday");
        break;
    default:
        Console.WriteLine("Another day");
        break;
}

🧠 Logic Insight

  • if is great for range-based logic
  • switch is better for exact value choices
  • Always close with else or default if needed

🧪 Try This

  • Ask the user for a number and print whether it’s positive, negative, or zero
  • Ask for a number 1–7 and use switch to print weekday
