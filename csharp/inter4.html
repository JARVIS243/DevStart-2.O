C# – Intermediate 4: Static Members

Learn how to use static variables and methods in C#

📌 What is Static?

Static means shared – belongs to the class, not individual objects.

You can use static for:

  • Variables shared by all objects
  • Methods that can be called without creating an object

🧮 Static Variable


class Counter
{
    public static int count = 0;

    public Counter()
    {
        count++;
    }
}

// Usage
Counter c1 = new Counter();
Counter c2 = new Counter();
Console.WriteLine(Counter.count);  // Output: 2

⚙️ Static Method


class MathUtils
{
    public static int Square(int x)
    {
        return x * x;
    }
}

// Call without object:
int result = MathUtils.Square(5);  // 25

🧠 Logic Tip

  • Use static for utility methods (like Math)
  • Don't use static when each object needs its own copy
  • static members are accessed using the class name
