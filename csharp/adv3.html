ğŸ§­ What is a Delegate?

A delegate is a type that holds a reference to a method. It's like a function pointer.

Delegates help you pass methods as arguments and are useful in event handling, callbacks, etc.

ğŸ“„ Delegate Example


using System;

delegate void GreetDelegate(string name);

class Program
{
    static void SayHello(string name)
    {
        Console.WriteLine("Hello " + name);
    }

    static void Main()
    {
        GreetDelegate greet = SayHello;
        greet("DevStart Learner");  // Output: Hello DevStart Learner
    }
}
