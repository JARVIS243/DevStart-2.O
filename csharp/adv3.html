C# – Advanced 3: Delegates

Understand and use delegates to point to methods

🧭 What is a Delegate?

A delegate is a type that holds a reference to a method. It's like a function pointer.

Delegates help you pass methods as arguments and are useful in event handling, callbacks, etc.

📄 Delegate Example


using System;

delegate void GreetDelegate(string name);

class Program
{
    static void SayHello(string name)
    {
        Console.WriteLine("Hello " + name);
    }

    static void Main()
    {
        GreetDelegate greet = SayHello;
        greet("DevStart Learner");  // Output: Hello DevStart Learner
    }
}

📌 Why Use Delegates?

  • To call methods dynamically
  • To implement callbacks
  • To write flexible and reusable code

🧠 Logic Tips

  • Delegate signature must match the method it points to
  • You can use multiple methods with one delegate (multicast)
  • Useful with events and LINQ
