Run non-blocking tasks using async programming

‚ö° What is Async Programming?

Async lets your program continue running while waiting for tasks like file read, web download, or delay.

Use async and await keywords to implement it easily.

üìÑ Example ‚Äì Async Method


using System;
using System.Threading.Tasks;

class Program
{
    static async Task DoTask()
    {
        Console.WriteLine("Task started...");
        await Task.Delay(2000); // Wait 2 seconds
        Console.WriteLine("Task completed.");
    }

    static void Main()
    {
        Task t = DoTask();
        Console.WriteLine("Main method continues...");
        t.Wait(); // Wait for async task to finish
    }
}

üß† Logic Tips

  • async keyword marks a method as asynchronous
  • await pauses execution until the task finishes
  • Use Task or Task<T> return types
  • Don't block main thread with long operations
