C# – Advanced 7: Multithreading

Run multiple tasks in parallel using threads

⚙️ What is Multithreading?

Multithreading lets your program run multiple tasks at once, improving performance for long or parallel operations.

📄 Basic Thread Example


using System;
using System.Threading;

class Program
{
    static void PrintNumbers()
    {
        for (int i = 1; i <= 5; i++)
        {
            Console.WriteLine("Number: " + i);
            Thread.Sleep(500); // Sleep for 0.5 seconds
        }
    }

    static void Main()
    {
        Thread t = new Thread(PrintNumbers);
        t.Start();

        Console.WriteLine("Main thread is free to do other work...");
    }
}

🧠 Logic Tips

  • Thread class is used to create and start a thread
  • Start() begins thread execution
  • Sleep(ms) pauses thread temporarily
  • Always handle thread safety when accessing shared data

🧪 Try This

  • Create two threads: one prints numbers, the other prints letters
  • Start both threads in Main()
  • Use Thread.Sleep() to simulate delays
