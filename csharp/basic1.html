C# – Basic 1: Introduction & Hello World

Get started with C# programming and write your first program!

👋 What is C#?

C# (pronounced "C-Sharp") is a modern, object-oriented programming language developed by Microsoft. It's used for building desktop apps, games (Unity), and enterprise software.

🛠 Setting Up

  • Install Visual Studio or Visual Studio Code
  • Install the .NET SDK from dotnet.microsoft.com
  • To run C# code: open terminal and type dotnet new console

✅ Your First Program


using System;

class Program
{
    static void Main(string[] args)
    {
        Console.WriteLine("Hello, DevStart!");
    }
}

This prints: Hello, DevStart!

🧠 Logic First

Even though this is a simple program, it shows structure:

  • using System; – includes input/output features
  • Main – starting point of the program
  • Console.WriteLine – prints to screen

🧪 Try This

  • Change the message and print your name
  • Add another Console.WriteLine() line below
  • Explore what happens if you remove ;
