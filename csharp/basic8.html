C# – Basic 8: Methods and Functions

Organize your code using reusable blocks called methods (functions)

🧩 What is a Method?

A method is a reusable block of code that performs a task.

📌 Defining a Method


void GreetUser()
{
    Console.WriteLine("Welcome to DevStart!");
}

void means it doesn't return any value.

📞 Calling a Method


GreetUser();  // Output: Welcome to DevStart!

📥 Method with Parameters


void SayHello(string name)
{
    Console.WriteLine("Hello, " + name);
}

// Call:
SayHello("Aju");

🔁 Method with Return


int Add(int a, int b)
{
    return a + b;
}

int result = Add(3, 4);
Console.WriteLine("Sum: " + result);

🧠 Logic Tip

  • Use methods to organize and reuse logic
  • Always match parameter types when calling
  • Use return values when you need results from a method
