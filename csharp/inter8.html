C# – Intermediate 8: Indexers

Access elements in a class using array-like syntax

📦 What is an Indexer?

An indexer lets you access instances of a class like arrays, using square brackets.

🔢 Example


class Numbers
{
    private int[] data = new int[5];

    public int this[int index]
    {
        get { return data[index]; }
        set { data[index] = value; }
    }
}

// Usage
Numbers n = new Numbers();
n[0] = 100;
n[1] = 200;
Console.WriteLine(n[0]);  // Output: 100

🧠 Logic Tip

  • Indexers use this keyword with square brackets
  • Useful for wrapping arrays or list-like behavior in custom classes
