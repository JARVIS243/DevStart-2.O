C# – Basic 2: Variables & Data Types

Store and manage values using variables and data types

🔤 What is a Variable?

A variable is like a container to store data that your program can use and change.

🧱 Declaring Variables


int age = 20;
string name = "Aju";
double pi = 3.14;
bool isStudent = true;
  • int – whole numbers
  • string – text
  • double – decimal numbers
  • bool – true or false

📥 Input & 📤 Output


using System;

class Program
{
    static void Main()
    {
        Console.Write("Enter your name: ");
        string userName = Console.ReadLine();
        Console.WriteLine("Hello, " + userName + "!");
    }
}

🧠 Logic Focus

Understand how data is stored, labeled, and reused in your code.

  • Use meaningful variable names
  • Remember to match type with value (e.g., don’t assign text to an int)

🧪 Try This

  • Create variables for your name, age, and favorite number
  • Use Console.WriteLine() to display them
  • Try changing the values and see the output
