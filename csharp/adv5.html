ğŸ” What is LINQ?

LINQ (Language Integrated Query) lets you query collections like arrays, lists, or databases using readable syntax.

ğŸ“„ Basic LINQ Query


using System;
using System.Linq;

class Program
{
    static void Main()
    {
        int[] numbers = { 5, 10, 15, 20, 25 };

        var result = from n in numbers
                     where n > 10
                     select n;

        foreach (int num in result)
        {
            Console.WriteLine(num);
        }
    }
}

20
25
