C# – Advanced 5: LINQ Basics

Use Language Integrated Query (LINQ) to query data easily

🔍 What is LINQ?

LINQ (Language Integrated Query) lets you query collections like arrays, lists, or databases using readable syntax.

📄 Basic LINQ Query


using System;
using System.Linq;

class Program
{
    static void Main()
    {
        int[] numbers = { 5, 10, 15, 20, 25 };

        var result = from n in numbers
                     where n > 10
                     select n;

        foreach (int num in result)
        {
            Console.WriteLine(num);
        }
    }
}

Output:


15
20
25

⚙️ Method Syntax


var result = numbers.Where(n => n > 10);

This is equivalent to the query syntax using lambda expressions.

🧠 Logic Tips

  • Use LINQ for filtering, sorting, selecting, grouping, etc.
  • LINQ works with arrays, lists, and even databases (with LINQ to SQL)
  • Method syntax and query syntax are interchangeable
