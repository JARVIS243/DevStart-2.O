C# – Basic 3: Input and Output

Learn how to read from the user and display information

📥 Input

Use Console.ReadLine() to take user input.


Console.Write("Enter your city: ");
string city = Console.ReadLine();
Console.WriteLine("You live in " + city);

📤 Output

Use Console.WriteLine() and Console.Write() to print values.


Console.Write("Name: ");
string name = Console.ReadLine();

Console.WriteLine("Welcome, " + name + "!");

🔁 Converting Input Types

All input is read as string. Convert it to numbers using parsing methods:


Console.Write("Enter your age: ");
int age = int.Parse(Console.ReadLine());
Console.WriteLine("Next year you will be " + (age + 1));

🧠 Logic Boost

  • Input + processing + output = program structure
  • Always validate and convert user input carefully
