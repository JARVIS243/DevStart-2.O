C# – Basic 7: Arrays and Strings

Learn how to store and work with multiple values in C#

📦 Arrays

An array stores multiple values of the same type.


int[] numbers = { 10, 20, 30, 40 };

for (int i = 0; i < numbers.Length; i++)
{
    Console.WriteLine(numbers[i]);
}
  • [] indicates an array
  • .Length returns number of elements

🧵 Strings

Strings are sequences of characters. Common methods:


string name = "DevStart";

Console.WriteLine(name.Length);       // 8
Console.WriteLine(name.ToUpper());    // DEVSTART
Console.WriteLine(name.ToLower());    // devstart
Console.WriteLine(name.Contains("Start")); // true

🔁 foreach Loop (for Arrays)


string[] langs = { "C#", "Java", "Python" };

foreach (string lang in langs)
{
    Console.WriteLine("I like " + lang);
}

🧠 Logic Skills

  • Arrays are useful for handling groups of data
  • Strings behave like arrays of characters
  • foreach simplifies iteration
