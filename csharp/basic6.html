C# – Basic 6: Loops

Repeat tasks using loops in C# – for, while, do-while

🔁 Why Loops?

Loops allow you to repeat a block of code multiple times until a condition is met.

🔹 for Loop


for (int i = 1; i <= 5; i++)
{
    Console.WriteLine("Count: " + i);
}

🔸 while Loop


int x = 1;
while (x <= 3)
{
    Console.WriteLine("While Loop: " + x);
    x++;
}

🔁 do-while Loop


int y = 1;
do
{
    Console.WriteLine("Do-While Loop: " + y);
    y++;
} while (y <= 2);

🧠 Logic Tip

  • Use for when you know how many times to loop
  • while checks before running; do-while checks after
  • Be careful to avoid infinite loops!

🧪 Try This

  • Print numbers from 1 to 10 using a for loop
  • Use a while loop to print "Hello" 5 times
  • Use do-while to ask a user password until it's correct
