C# – Advanced 9: Advanced Collections

Explore powerful collections like Dictionary, HashSet, and Queue

🧺 Dictionary<TKey, TValue>

Stores key-value pairs for fast lookup.


Dictionary<string, int> scores = new Dictionary<string, int>();
scores["Aju"] = 95;
scores["Dev"] = 88;

Console.WriteLine(scores["Aju"]); // 95

🔁 Queue<T>

First-In-First-Out (FIFO) structure.


Queue<string> tasks = new Queue<string>();
tasks.Enqueue("Task1");
tasks.Enqueue("Task2");

Console.WriteLine(tasks.Dequeue()); // Task1

🚫 HashSet<T>

Stores unique elements only (no duplicates).


HashSet<int> ids = new HashSet<int>();
ids.Add(101);
ids.Add(102);
ids.Add(101); // Ignored

Console.WriteLine(ids.Count); // 2

🧠 Logic Tips

  • Dictionary is fast for lookups using keys
  • Queue is ideal for task processing or lines
  • HashSet is perfect for uniqueness and set operations

🧪 Try This

  • Create a dictionary of usernames and passwords
  • Use a queue to simulate a print queue
  • Use a hashset to collect unique numbers from a list
