TypeScript – Intermediate 4: Generics

Write reusable and flexible components with generics

📦 What Are Generics?

Generics let you write functions, classes, or interfaces that work with any data type.


// Generic function
function identity(value: T): T {
  return value;
}

console.log(identity("Aju")); // Aju
console.log(identity(101));   // 101

📚 Generic with Arrays


function logItems(items: T[]): void {
  items.forEach(item => console.log(item));
}

logItems(["apple", "banana", "cherry"]);
logItems([10, 20, 30]);
