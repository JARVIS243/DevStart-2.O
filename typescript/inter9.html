TypeScript – Intermediate 9: DOM Manipulation

Access and manipulate HTML elements safely using TypeScript

🌐 Access DOM Elements

TypeScript adds type safety when selecting and using DOM elements.


// HTML:
// <input id="username" type="text">
// <button id="btn">Click</button>

const input = document.getElementById("username") as HTMLInputElement;
const btn = document.getElementById("btn") as HTMLButtonElement;

btn.addEventListener("click", () => {
  alert("Hello " + input.value);
});

✅ Type Assertions

Use as HTMLInputElement to tell TS the exact element type.


const title = document.querySelector("h1") as HTMLHeadingElement;
title.style.color = "blue";
