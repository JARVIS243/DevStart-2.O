TypeScript – Basic 3: Arrays and Tuples

Store multiple values using arrays and tuples

📦 Arrays


let scores: number[] = [80, 90, 100];
let names: string[] = ["Aju", "Anu", "Asha"];
console.log(scores[0]); // 80

🔗 Tuples

Tuples can hold a fixed number of elements with different types.


let student: [string, number] = ["Aju", 21];
console.log(student[0]); // Aju
