TypeScript – Advanced 9: Using TypeScript with React

Learn how TypeScript improves safety and structure in React apps

⚛️ Functional Component with Props


type Props = {
  name: string;
  age?: number;
};

function Greeting({ name, age }: Props) {
  return <h1>Hello, {name}!</h1>;
}

Note: Use .tsx for React components with TypeScript.

🧩 useState with Type


import { useState } from "react";

const [count, setCount] = useState<number>(0);

📋 useRef with DOM


import { useRef } from "react";

const inputRef = useRef<HTMLInputElement>(null);

🧪 Try This

  • Create a Button component with typed props
  • Use useState with a boolean value
  • Use useRef to focus an input field

⚙️ Setup Tips

  • Install: npx create-react-app my-app --template typescript
  • Make sure all React components use .tsx extension
  • Use VS Code for intellisense and linting
