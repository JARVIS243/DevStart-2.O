TypeScript – Basic 2: Number, String, Boolean

Learn the three core types used in every TypeScript program.

🔢 Number

Used for all numeric values.


let age: number = 25;
let price: number = 99.99;
console.log(age, price);

🔤 String

Text data enclosed in quotes.


let name: string = "Aju";
let greeting: string = `Hello, ${name}`;
console.log(greeting);

✅ Boolean

True/false values for conditions.


let isOnline: boolean = true;
let isFinished: boolean = false;
console.log(isOnline, isFinished);

🧪 Try This

  • Create a variable for your name (string)
  • Set your age (number)
  • Indicate if you are a student (boolean)
  • Print a message using all three values
