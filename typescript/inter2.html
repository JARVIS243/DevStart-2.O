Build class hierarchies and protect data using public, private, and protected

üìò Inheritance

Use extends to create a subclass from a base class.


class Animal {
  name: string;

  constructor(name: string) {
    this.name = name;
  }

  move(): void {
    console.log(`${this.name} moves.`);
  }
}

class Dog extends Animal {
  bark(): void {
    console.log(`${this.name} barks.`);
  }
}

const dog = new Dog("Tommy");
dog.move(); // Tommy moves.
dog.bark(); // Tommy barks.

üîê Access Modifiers

Control who can access properties and methods:

  • public ‚Äì accessible everywhere (default)
  • private ‚Äì accessible only inside the class
  • protected ‚Äì accessible in class and subclasses

class BankAccount {
  public name: string;
  private balance: number;

  constructor(name: string, balance: number) {
    this.name = name;
    this.balance = balance;
  }

  getBalance(): number {
    return this.balance;
  }
}

const acc = new BankAccount("Aju", 5000);
console.log(acc.name); // Aju
// console.log(acc.balance); // ‚ùå Error: private
console.log(acc.getBalance()); // 5000
