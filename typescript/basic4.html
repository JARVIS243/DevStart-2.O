TypeScript – Basic 4: Enums

Use enums to define a set of named constants

🎯 Basic Enum


enum Direction {
  Up,
  Down,
  Left,
  Right
}

let move: Direction = Direction.Left;
console.log(move); // 2

📌 Custom Values


enum Status {
  Pending = 1,
  InProgress = 3,
  Done = 5
}
