TypeScript – Advanced 2: Mapped Types

Transform and generate new types dynamically using keys of other types

🔁 What Are Mapped Types?

Mapped types let you create new types by looping over keys of an existing type.


type Person = {
  name: string;
  age: number;
};

// Make all fields optional
type OptionalPerson = {
  [K in keyof Person]?: Person[K];
};

🔒 Make All Fields Readonly


type ReadonlyPerson = {
  [K in keyof Person]: Readonly<Person[K]>;
};

🔧 Use with Utility Types

Mapped types are used behind the scenes in utility types like:

  • Partial<T>
  • Readonly<T>
  • Pick<T, K>
  • Record<K, T>

🧪 Try This

  • Define a type Student with 2–3 fields
  • Create OptionalStudent using a mapped type
  • Create ReadonlyStudent the same way
  • Compare with Partial<Student>
