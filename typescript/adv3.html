TypeScript – Advanced 3: Conditional Types

Build flexible types using conditional logic in type definitions

🤔 What Are Conditional Types?

Conditional types let you define types based on a condition.


type IsString = T extends string ? "yes" : "no";

type A = IsString<string>;  // "yes"
type B = IsString<number>;  // "no"

🔗 Distributive Conditional Types

When used with union types, conditional types are applied to each part.


type ToArray = T extends any ? T[] : never;

type StrArr = ToArray<string>;    // string[]
type UnionArr = ToArray<string | number>; // string[] | number[]
