TypeScript – Advanced 8: Advanced Generics

Use constraints, defaults, and multiple generic parameters

🔁 Generics Recap

Generics let you write flexible and reusable code.


function identity(value: T): T {
  return value;
}

🔒 Constraints

Limit what types can be passed using extends.


function getLength(item: T): number {
  return item.length;
}

getLength("hello"); // ✅
getLength([1, 2, 3]); // ✅
// getLength(123); ❌ Error

🌐 Default Generic Types


type ApiResponse = {
  data: T;
  status: number;
};

const res: ApiResponse = {
  data: "Success",
  status: 200
};

🔗 Multiple Type Parameters


function pair(key: K, value: V): [K, V] {
  return [key, value];
}

const result = pair("id", 123); // ["id", 123]

  • Use default types for a generic type
  • Make a type or function that uses two generic parameters
