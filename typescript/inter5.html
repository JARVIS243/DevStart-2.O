Modules allow you to split your code into separate files. You export things from one file and import them in another.


// mathUtils.ts
export function add(a: number, b: number): number {
  return a + b;
}

// main.ts
import { add } from "./mathUtils";
console.log(add(5, 3)); // 8

Make sure to compile with module support: tsc --module commonjs main.ts

Namespaces group related logic in the same file using the namespace keyword.


namespace Utils {
  export function greet(name: string): void {
    console.log(`Hello, ${name}`);
  }
}

Utils.greet("Aju"); // Hello, Aju
