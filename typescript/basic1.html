TypeScript – Basic 1: Introduction

What is TypeScript and why should you learn it?

🚀 What is TypeScript?

TypeScript is a strongly-typed superset of JavaScript that compiles to plain JavaScript.

  • It adds type safety
  • Helps catch errors during development
  • Makes code more readable and maintainable

🔧 How to Use It

npm install -g typescript
// hello.ts
let message: string = "Hello, TypeScript!";
console.log(message);

Compile it: tsc hello.ts
