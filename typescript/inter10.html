TypeScript – Intermediate 10: tsconfig and Compilation

Learn how to set up TypeScript projects using tsconfig.json

🛠️ Creating tsconfig.json

Initialize a TypeScript project using:

tsc --init

This generates a tsconfig.json file to configure how TS compiles.

🔧 Common tsconfig Options


{
  "compilerOptions": {
    "target": "es6",
    "module": "commonjs",
    "strict": true,
    "outDir": "./dist",
    "rootDir": "./src"
  }
}
  • target – JavaScript version to compile to
  • module – module system (e.g., commonjs)
  • strict – enable strict type checking
  • outDir – where compiled files go
  • rootDir – source folder for TS files

📁 Folder Structure

project/
├── src/
│   └── index.ts
├── dist/
├── tsconfig.json

Compile using:

tsc

🧪 Try This

  • Initialize a new TypeScript project with tsc --init
  • Create a src/index.ts file
  • Configure tsconfig.json to use outDir and rootDir
  • Compile and run the JS output
