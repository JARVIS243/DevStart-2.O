TypeScript – Advanced 7: Advanced Interfaces & Inheritance

Extend interfaces and combine types for powerful object structures

🧩 Interface Inheritance

Interfaces can extend other interfaces to create complex hierarchies.


interface Animal {
  name: string;
}

interface Dog extends Animal {
  bark(): void;
}

const myDog: Dog = {
  name: "Tommy",
  bark() {
    console.log("Woof!");
  }
};

🔗 Multiple Interface Inheritance


interface Flyer {
  fly(): void;
}

interface Swimmer {
  swim(): void;
}

interface Duck extends Flyer, Swimmer {}

const duck: Duck = {
  fly() {
    console.log("Flying...");
  },
  swim() {
    console.log("Swimming...");
  }
};

🔄 Interface Merging

TypeScript automatically merges interfaces with the same name.


interface Person {
  name: string;
}

interface Person {
  age: number;
}

const p: Person = { name: "Aju", age: 22 };

  • Make an object that satisfies the Employee interface
  • Use interface merging to add more properties to Employee
