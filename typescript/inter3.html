TypeScript – Intermediate 3: Interfaces with Classes

Use interfaces to define a contract for your classes

📄 Interface in Class

A class can implement an interface to follow a specific structure.


interface Printable {
  print(): void;
}

class Invoice implements Printable {
  constructor(private amount: number) {}

  print(): void {
    console.log(`Invoice amount: ₹${this.amount}`);
  }
}

const bill = new Invoice(2500);
bill.print(); // Invoice amount: ₹2500

👥 Multiple Interfaces


interface Printable {
  print(): void;
}

interface Savable {
  save(): void;
}

class Report implements Printable, Savable {
  print() {
    console.log("Printing Report...");
  }

  save() {
    console.log("Saving Report...");
  }
}
