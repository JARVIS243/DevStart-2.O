TypeScript – Basic 6: Type Aliases & Interfaces

Define your own types using alias and interface.

✏️ Type Alias


type User = {
  name: string;
  age: number;
};

let user1: User = { name: "Aju", age: 22 };

📘 Interface


interface Student {
  id: number;
  enrolled: boolean;
}

let s1: Student = { id: 1, enrolled: true };
