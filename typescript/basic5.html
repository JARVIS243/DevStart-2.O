TypeScript – Basic 5: Functions

Learn to define and call functions with typed parameters

📘 Function Example


function greet(name: string): string {
  return `Hello, ${name}`;
}
console.log(greet("Aju"));

🧮 Default and Optional Parameters


function add(a: number, b: number = 10): number {
  return a + b;
}
console.log(add(5)); // 15
